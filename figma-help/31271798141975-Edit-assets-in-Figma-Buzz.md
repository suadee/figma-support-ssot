## 내용

Who can use this feature

Who can use this feature

While in beta, Figma Buzz is available on all seats and plans.

Anyone with `can edit` permissions on a file can edit assets

There are two ways to edit assets in Figma Buzz. You can edit content (like text or images) directly in  **Bulk edit**. You can also edit the visual design of an asset on the canvas in  **Asset view**, including fill, border, and layout.

## Edit content in Bulk edit

The **Bulk edit** tab shows all the assets in your file as rows, with each editable property as a column. It's the fastest way to update content across many assets at once.

To open start editing, click the  **Bulk** **edit** icon from the left sidebar.

![Select bulk edit to edit content of assets in a table view..png](https://help.figma.com/hc/article_attachments/40665352555415)

**Note**: **Bulk edit** replaces the **Edit content** panel from earlier versions of Figma Buzz. If you previously used the **Edit content** panel to update text and image fields, you'll find the same functionality in **Bulk edit**.

When you click a row, an asset preview appears on the right. To give yourself more room to work, you can hide columns you don't need—click  **Settings** to toggle individual columns on or off.

### Edit a cell

Double-click any cell to start editing. How editing works depends on the cell type:

- **Text:** Type directly in the cell. The asset updates in the preview as you type.
- **Media:** An inline toolbar appears with options to crop, adjust, or replace the image. Hover over an image cell to reveal the  **Change image** icon to upload a new image.
- **Component properties/Variable modes:** For [assets built with components or variables from your design system](https://help.figma.com/hc/en-us/articles/35659298120215-Build-configurable-Figma-Buzz-templates-with-your-design-system#h_01K7MP9Q14K3RYJDEZJN3RMJ38), a dropdown appears with the available values or modes for that property. For boolean properties (like show/hide), click to toggle the value on or off.

To navigate between cells, use the arrow keys. Press `Enter` to confirm your change and close the cell editor.

**Note**: You can only edit a cell if the corresponding layer isn't [locked in the template](https://help.figma.com/hc/en-us/articles/31271689852823-Find-and-use-Figma-Buzz-templates#remove-guidelines). If a cell is grayed out or not editable, the template designer may have restricted that field.

### Edit multiple assets at once

You can update the same property across multiple assets at the same time using one of the following methods:

- **Select a range of cells**: Click a cell, then hold shift to select cells across multiple rows. Any edit you make applies to all selected cells.
- **Drag to fill**: To copy a cell's value down to the cells below it, select the cell and drag the small circle handle at the bottom corner of the cell downward. The value fills into each cell as you drag.
- **AI fill**: To have Figma AI generate contextually varied content across a range of cells, select a cell and drag the oval handle on the bottom of the cell; AI fill generates unique values for each cell based on the content of the original.

### Resize assets

To resize assets, see [Resize assets in Figma Buzz](https://help.figma.com/hc/en-us/articles/34324344840855-Resize-assets-in-Figma-Buzz).

## Edit assets on the canvas

For changes to an asset's visual design (such as background, border, layout, or adding new elements), you can work directly on the canvas in  **Asset view**.

To open **Asset view**, click the  **Asset view** icon from the left sidebar.

Assets on the canvas have either a blue outline or a pink outline:

- **Blue outline:** No template guidelines. You can edit the asset freely.
- **Pink outline:** Template guidelines are active. You can only edit unlocked fields.

[Learn more about template guidelines](https://help.figma.com/hc/en-us/articles/31271689852823-Find-and-use-Figma-Buzz-templates#remove-guidelines).

### Size

Select an asset, then open the **Size** dropdown in the toolbar to choose a preset size or enter custom dimensions.

[Learn more about resizing assets in Figma Buzz](https://help.figma.com/hc/en-us/articles/34324344840855-Resize-assets-in-Figma-Buzz).

### Fill

Click the color swatch in the toolbar to edit an asset's background:

- **Color:** Choose from existing file colors or open the color picker.
- **Gradient:** Add a gradient and click **Adjust** to customize colors and spread.
- **Image & video:** Upload a file from your device or [generate one with Figma AI](https://help.figma.com/hc/articles/31434032783383).

### Border

Select an asset and choose a solid or dashed **Border** from the inline toolbar.

### Add text, images, and more

In **Asset view**, use the tools in the left navigation bar to add new elements to an asset:

- **Text:** Click the text tool or press `T`. Edit font, size, color, alignment, and effects from the properties panel.
- **Media:** Upload from your device, insert stock photos from Unsplash, or [generate an image with Figma AI](https://help.figma.com/hc/articles/31434032783383). [Learn more about adding and editing images](https://help.figma.com/hc/articles/34290428383127). Also [supports video](https://help.figma.com/hc/en-us/articles/33799379167767-Use-videos-in-Figma-Buzz).
- **Inserts:** Add shapes or components from your team libraries. [Learn more about using libraries in Figma Buzz](https://help.figma.com/hc/articles/33799043961879).

To edit text or media, select the object on the asset, and use the inline toolbar to edit.

## Use Figma Design tools

Toggle the **Design** button in the toolbar to access the full Figma Design toolset for advanced edits.

[Learn more about using Figma Design tools in Figma Buzz](https://help.figma.com/hc/articles/31271851622807).
