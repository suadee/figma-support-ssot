## 내용

Before you start

Who can use this feature

Available on [any plan](https://help.figma.com/hc/en-us/articles/360040328273)

Applies to all file types

Files in Figma are organized into projects. Each project has its own set of permissions, so moving a file may change who has access to it.

Here are some things to consider when moving a file:

| **Action** | **Access you'll need** | **Who will have access to the file** |
| --- | --- | --- |
| Move a file from your **drafts** into a project | - You must have `can edit` permissions in the destination project - You must have a [seat type](https://help.figma.com/hc/en-us/articles/360039960434) that gives you full access to the file type you want to move. (For example, to move a design file, you need a Full seat. To move a FigJam file, you need either a Full, Dev, or Collab seat). | - People invited to the file - Everyone in the destination project |
| Move a file between projects in the **same** team | You must have `can edit` permissions in the current and destination projects | - People invited to the file - Everyone in the destination project |
| Move a file to a project in a **different** team | Before you can move a file to a project in a different team, you need to:   - Join both teams - Have `can edit` permissions in both teams | - People invited to the file - Everyone in the destination project - Everyone in the new team - Enterprise only: Everyone in the destination workspace, if the [workspace is set as an audience](https://help.figma.com/hc/en-us/articles/360040531773-Share-files-and-prototypes) |

Move a file from the file browser Move a file from the canvas

1. In the file browser, find the file you’d like to move.
2. Right click the file thumbnail and select **Move file…**
3. Search for a project or choose from the default options.
4. Click **Move.**

To move multiple files at once:

1. Hold `Command` (Mac) or `Control` (PC) while selecting multiple files.
2. Right-click and select **Move [X] Files**.

You can only move multiple files at once if you have the required access in each selected file.

1. Open the file you’d like to move.
2. Click an empty spot on the canvas to deselect any layers.
3. Click the  icon next to the file name and select **Move…**
4. Search for a project or choose from the default options.
5. Click **Move.**

**Tip**: If you don’t see the project where you’d like to move the file, you might not have `can edit` permissions in the destination project.

## Move a file between Figma accounts or organizations

It’s not possible to move a file between different Figma accounts or organizations using the steps above. However, you can:

- Move one or more files into a project and then [transfer the project](https://help.figma.com/hc/en-us/articles/10250387712407) to another account. This requires both the sender and the receiver to be on a [paid plan](https://help.figma.com/hc/en-us/articles/360040328273).
- [Save a copy of a file to your computer](https://help.figma.com/hc/en-us/articles/8403626871063-Save-a-local-copy-of-a-file) and [import it](https://help.figma.com/hc/en-us/articles/360041003114) to a different account. If the file uses styles or components from a [team library](https://help.figma.com/hc/en-us/articles/360041051154), it won’t receive any library updates after it is imported into a different account.
