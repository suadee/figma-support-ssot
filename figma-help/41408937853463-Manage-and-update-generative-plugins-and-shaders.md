## 내용

**Generated plugins and shaders are currently in open beta.** Features may change and you may experience performance issues as we continue to improve generated plugins and shaders. Learn more about [what's included in the beta](https://help.figma.com/hc/en-us/articles/4406787442711-What-Figma-features-are-in-beta).

Before you Start

Who can use this feature

A generative shader or plugin can only be edited or deleted by its ownerplugin can be done by its owner

You can manage generative tools from any Figma Design file you have access to

Figma uses WebGL to render shaders. Make sure your [system and browser meet the minimum requirements](https://help.figma.com/hc/en-us/articles/360039827194-What-are-the-system-requirements-for-Figma).

Make edits to, rename, or delete your generative plugins and shaders.

**New to generative plugins and shaders?** Check out our [quick start guide](https://help.figma.com/hc/articles/41147702210071/) to learn more.

## Make edits

Update your tool from the agent or from the **Tools** panel of any Figma Design file.

1. Open any Figma Design file.
2. Click  **Agents** in the navigation bar on the left.
3. In the prompt box, type forward slash `/` and select the tool you want to edit from the menu.
4. Describe the edits you’d like to make, like `Add a control that...` .
5. Press `Enter` to submit your prompt.
6. Test your tool and iterate as needed.

**Tip**: Test things often, and test it on various places. If using real designs, make sure to use a duplicate of the design to preserve the original asset.

[Get more prompting tips.](https://help.figma.com/hc/en-us/articles/41408020198935)

## Rename

To rename a generated plugin or shader:

1. Click the **Tools** tab in the navigation bar.
2. Use the search bar to find your tool by name. You can also use the **Source** dropdown to filter by **Created by you**.
3. Hover over your plugin and click the  more options icon, then select **Rename**.

![rename-tool.png](https://help.figma.com/hc/article_attachments/41456895145367)

## Delete

To delete a generated plugin or shader:

1. Click the **Tools** tab in the navigation bar.
2. Use the search bar to find your tool by name. You can also use the **Source** dropdown to filter by **Created by you**.
3. Hover over your plugin and click the  more options icon, then select **Delete.**
4. From the modal, click **Delete**.
