## 내용

Who can use this feature

Available on the [Organization and Enterprise plans](https://help.figma.com/hc/en-us/articles/360040328273)

Everyone except guests

Plugins and widgets are third-party applications that run in Figma's products. These tools can access information about you and your files, so your company may want to review and approve them before making them available to use.

If your organization requires approvals, you’ll need to request access to each plugin or widget that has not been approved.

**Note**: On the Enterprise plan, organization admins can approve plugins and widgets on a workspace-by-workspace basis. As a result, you might only be able to use a plugin or widget in one workspace, but not another.

### Make a request

![Request widget approval dialog showing a note area with user rationale for widget use. Options to cancel or request approval.](https://help.figma.com/hc/article_attachments/31443503765143)

When making a request, we recommend giving the reviewer some context to help them with their decision. All organization admins will get notified when you send a request. To make a request:

Figma Community In a file

1. In [Figma Community](https://www.figma.com/community), navigate to the plugin or widget you’d like to use.
2. Click **Open in.**
3. Select **New file**.
4. Provide a reason and click **Request**.

1. From within a file, open **Actions** from the toolbar.
2. Select the **Plugins** **& widgets** tab.
3. Search for the name of the plugin or widget you’d like to use.
4. Click on the search result.
5. Click **Request approval**.
6. Provide a reason and click **Request**.

### What happens if my request is declined?

You’ll get [notified in Figma](https://help.figma.com/hc/en-us/articles/360039813234) and via email if your request is declined. From the notification, you can re-submit the request with more information, if needed.
