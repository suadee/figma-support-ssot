## 내용

Before you start

Who can use this feature

 

Supported on [any team or plan](https://help.figma.com/hc/en-us/articles/360040328273-Choose-a-Figma-Plan)

Anyone can rename their own files in their drafts

Only team admins or members with edit access can rename team files

Rename files and projects to explain their purpose or contents. Set a file or project name when you first create it, and update it as the project evolves.

## Rename a file

Figma names every new file you create as **Untitled**. You can change this to something that better explains the contents of the file.

File names don't need to be unique, but unique names make it easier to search for and find the correct file.

You can update the file name the following ways:

From the file browser In a file

1. Right-click the file in the file browser.
2. Select **Rename** from the file menu.

1. Figma displays the file's location and name at the top of the left sidebar.
2. Click on the existing file name to edit the name.
3. Click outside of the field or press `Enter` to apply your changes.

## Rename a project

1. Open the project page in the file browser.
2. Right-click the project folder > **Settings**.
3. Rename your project.
