## 내용

Before you Start

Who can use this feature

Resource settings for Make kits are available on [Organization and Enterprise plans](https://help.figma.com/hc/en-us/articles/360040328273)

Organization admins can adjust resource settings

Make kits allow members of your organization to jump straight into building designs and prototypes in Make that are realistic to how your product looks and functions without having to add code and style context from scratch.

Make kits can include:

- npm packages for code context
- Variables and styles from your Figma Design library
- Guidelines to help Figma Make understand how to use your system's assets

Once a Make kit is assembled, it can be published to the organization so that members can use it their Figma Make projects to start building faster.

[Learn more about Make kits.](https://help.figma.com/hc/articles/39313426360087)

Admins of an organization can surface Make kits to the entire organization so teams know which kit are the best ones to work with. Admins can:

- [Approve a Make kit](#h_01KMNBSCG6F9K21WKD5MXJWT0Z)
- [Add a Make kit by default](#h_01KMNBSCG9AD7YCMJ7E4HYHWAB)

### Approve a Make kit

Approving a Make kit for an organization lets everyone know that the kit has been recommended by an admin. This helps people feel more comfortable choosing which kit to use in their Make projects.

When a Make kit is approved, a  check mark will appear next to the kit name wherever it appears in Make files in Admin settings. Approved Make kits are also pinned to the top of the list when selecting a kit to use in a project.

1. From the [file browser](https://help.figma.com/hc/en-us/articles/14381406380183), click  **Admin** in the left sidebar.
2. Select **Resources** on the left sidebar.
3. Select the **Make kits** tabs.
4. Search for—or navigate to—the kit you’d like to approve or unapprove.
5. Click  **More settings** on the right of the kit name and select **Set as approved** or **Remove from approved**.

### Add a Make kit by default

Enabling a Make kit by default adds the kit to Make files without having to manually search for and add the kit. This setting enables the kit for the entire organization.

1. From the [file browser](https://help.figma.com/hc/en-us/articles/14381406380183), click **Admin** in the left sidebar.
2. Select **Resources** on the left sidebar.
3. Select the **Make kits** tabs.
4. Search for—or navigate to—the kit you’d like to approve or unapprove
5. Click **More settings** on the right of the kit name and select **Add by default** or **Don’t add by default**.
