## About this video

In this Figma Tip, follow along with Designer Advocate Lauren as she breaks down how to create [code layers](https://help.figma.com/hc/en-us/articles/31242824165143) in Figma Sites.

Code layers are a powerful feature that lets you use AI-generated code to create unique and powerful experiences, no coding skills required.

We recommend watching this video even if you watched the [code layers lesson in the Figma Sites collection](https://help.figma.com/hc/en-us/articles/35895907782807), as Lauren showcases some unique and powerful use cases that weren't covered in depth in that lesson. Code layers are so flexible, that the more examples you learn from the more you realize the potential of what you can do with code layers.

[Learn more about code layers on the help center →](https://help.figma.com/hc/en-us/articles/31242824165143)
