## 내용

This feature has been developed by Flinto. For any questions regarding Flinto or the integration, please [contact the Flinto team](https://www.flinto.com/contact).

Flinto is a Mac app used by designers to create interactive and animated prototypes. [Flinto](https://www.flinto.com/) is only available for **Mac**. [Learn more about Flinto](https://www.flinto.com/learn).

Flinto gives you access to powerful tools for animated transitions, micro-interactions, scroll-based effects and more. With Flinto you can:

- Create custom animated transitions between Frames.
- Design your own micro-interactions with Flinto's **Behavior Designer**.
- Use a variety of touch gestures, including 3D touch.
- Add GIFs and videos to your prototypes.
- Export recording of your prototypes to share online.

The integration allows you to import your designs from Figma into Flinto. You can then turn them into animated and interactive prototypes.

Note: When importing Frames from Figma, Flinto will preserve most vector shapes and editable text layers. For any features that aren't supported, Flinto will flatten them into bitmap images.

# Connect Figma and Flinto

Before you can import your designs from Figma, you'll need to authorize the connection between Figma and Flinto.

This gives Flinto to access your Files, so they can import them into their app.

You'll only need to do this once, the first time you import a File from Figma.

1. Open the Flinto app.
2. Click **Import** in the toolbar. Or go to **Flinto > Import** in the menu bar. ![Flinto interface with an active toolbar showing options such as Insert, Group, and Import; document properties panel visible.](https://d33v4339jhl8k0.cloudfront.net/docs/assets/5aa962fe2c7d3a2c4983093d/images/5d51d18f0428631e94f948c8/file-jfrfWvlPNM.png)
3. Select the **Figma** tab: ![Figma integration panel in Flinto with "Sign Into Figma" button highlighted.](https://d33v4339jhl8k0.cloudfront.net/docs/assets/5aa962fe2c7d3a2c4983093d/images/5d51d16c2c7d3a68825e8d7f/file-Xg08r4rJnv.png)
4. Click the **Sign into Figma** button: ![Sign into Figma button highlighted in Flinto's import window for establishing connection.](https://d33v4339jhl8k0.cloudfront.net/docs/assets/5aa962fe2c7d3a2c4983093d/images/5d51d1782c7d3a68825e8d80/file-S4qrPIBbjM.png)
5. Your web browser will open Figma in a new tab. If you're not signed in, you will be prompted to do so now. Otherwise, click the **Allow Access** button: ![Flinto access request screen for Figma, showing options to allow or deny account access.](https://d33v4339jhl8k0.cloudfront.net/docs/assets/5aa962fe2c7d3a2c4983093d/images/5d51d1810428631e94f948c7/file-1aRjBni4u4.png)
6. Click the prompt to return to Flinto.

# Import a Design from Figma

Now that the connection is established, you can import your Figma Files into Flinto.

You'll be able to select from the Files you've most recently opened. If you haven't recently opened the File - or don't see the File you're looking for - try opening the File in Figma first.

1. In the Flinto app, click the **Import** button in the toolbar.
2. Select the **Figma** tab. You can then select which File you want to import into Flint. Your most recently edited Files are displayed at the top: ![Flinto import panel with Figma tab, showing design selection and iPhone XS screen size options.](https://d33v4339jhl8k0.cloudfront.net/docs/assets/5aa962fe2c7d3a2c4983093d/images/5d51d1cd0428631e94f948d1/file-2yxapauf2f.png)
3. Select the **Screen size** for the selected File. Frames. The default will be the size of the current document.
4. Click the **Import Frames** button.
5. Now you are ready to create your prototype in Flinto! Visit [Flinto’s learn page](https://www.flinto.com/learn) to learn more about getting started.

# Disconnect your Figma Account

1. Click the **Import** button in the toolbar to open the modal.
2. Select the **Figma** tab.
3. Click on your profile picture in the bottom left corner and select **Log Out**:![Figma import panel in Flinto showing file selection and screen size options, with a log out button highlighted.](https://d33v4339jhl8k0.cloudfront.net/docs/assets/5aa962fe2c7d3a2c4983093d/images/5d51d1f62c7d3a68825e8d8e/file-MTERvOW2Xi.png)
