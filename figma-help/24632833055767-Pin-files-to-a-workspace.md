## 내용

Who can use this feature

Available on the [Enterprise plan](https://help.figma.com/hc/en-us/articles/360040328273)

All members and admins of a workspace can pin files to it

Organization admins can pin files to any workspace

Pin files to a workspace to help people find the most important resources. For example, you might pin files related to:

- Your organization or department’s vision or mission
- Active goals and objectives
- Onboarding guides for new hires

Everyone in your organization can view pinned files in workspaces they have access to.

### What files can be pinned to a workspace?

![Share settings panel showing access permissions, view and edit options, and discoverable via search toggle in Figma.](https://help.figma.com/hc/article_attachments/24833140368407)

You can pin any file to its parent workspace, as long as the file’s permissions are set so anyone at your organization can view or edit it**.**

This includes files where the parent project or team has more restrictive, invite-only permissions.

If you change the permissions of a pinned file to **Only invited people**, Figma will automatically remove it from the pinned items.

## Pin a file to a workspace

![Pinned files section in a Figma workspace, showing various projects and an "Add file" option.](https://help.figma.com/hc/article_attachments/24833115600407)

**Note**: A maximum of eight files can be pinned to a workspace.

Pinned files appear at the top of a workspace, above the list of teams. When someone hovers over a pinned file, they get extra information about it, like: who pinned it and a description of why the file is important to people in the workspace.

When you pin a file, Figma notifies members and admins of the workspace via email and an in-app notification. If you pin multiple files at once, we’ll combine the notifications into a single message. People won’t get notified when a file is unpinned.

**Note:** When you pin a file, Figma makes it [discoverable via search](https://help.figma.com/hc/en-us/articles/360040531773) if that setting isn’t already turned on. This means members of your organization can find the file when searching from the file browser.

To pin a file:

1. From the [file browser](https://help.figma.com/hc/en-us/articles/14381406380183), click the name of your workspace in the left sidebar.
2. In the **Pinned** section at the top of the workspace, click **Add new**.
3. Select a file to pin.
4. Enter a description to explain why the file is important to people in the workspace. If the file isn’t shared with the organization, you’ll see a note that Figma will change the permissions on the file so anyone in the organization can access it.
5. Press **Pin file**.

**Tip:** You can also pin a file from anywhere in file browser, or from inside the file itself.

- From the file browser: Right click the file, then select **Pin to workspace**
- From inside a file: Click the  down arrow next to the file name and select **Pin to workspace**

## Remove a pinned file from a workspace

Only the person who pinned the file, and admins of the workspace, can unpin a file.

To unpin a file, right click it from anywhere in the file browser and select **Remove workspace pin**.
