## 내용

The Figma add-on for Google Workspace allows users to see previews of Figma and FigJam files (smart chips) in Google Docs, so teammates can easily reference the latest designs and project work in context.

## Get started

1. [Get the Figma add-on for Google Workspace →](https://workspace.google.com/marketplace/app/figma/1035183305161)
2. [Log in to your Figma account →](https://help.figma.com/hc/en-us/articles/360041064554-Log-in-or-add-accounts)
3. [Create a Google doc →](https://support.google.com/a/users/answer/9300311?hl=en)

## Add smart chips to a Google Doc

1. [Copy the link for a Figma or FigJam file →](https://help.figma.com/hc/en-us/articles/360040531773-Share-files-and-prototypes)
2. Open a new or existing Google Doc.
3. Paste the link in the Google Doc.
4. Press `Tab` or click on the prompt to see the Figma or FigJam file smart chip.
