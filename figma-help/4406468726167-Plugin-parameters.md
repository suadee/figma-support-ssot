## 내용

## Plugin parameters

Developers can create plugins with parameters which allow plugins to accept user input from the quick actions menu. Adding parameters to your plugin means you don't need to build a custom UI. It also provides a streamlined experience for users of the plugin, who can run a plugin directly from the quick actions menu. [Learn more about quick actions →](https://help.figma.com/hc/en-us/articles/360040328653)

### Example:

This plugin creates a shape on the canvas. It has no UI and can only be run via the quick actions menu (Mac) `⌘`+`/` (Windows) `Ctrl`+`/`. A user can select a shape type for the `Shape` parameter, then input number values for the `Width` and `Height` parameters to define the shape's dimensions.

![Quick actions menu displaying options for creating shapes with parameters: Rectangle, Ellipse, Star, Polygon.](https://help.figma.com/hc/article_attachments/4406476240663)

[Check out these plugins powered by your keyboard →](http://figma.com/c/collections/plugins-for-keyboard-users)

## Make plugins with parameters

Anyone can develop a plugin with parameters. If you're new to developing plugins, learn how to [make plugins for the Figma Community](https://help.figma.com/hc/en-us/articles/360039958874).

Figma's developer platform provides a detailed setup guide for the plugin API. If you're ready to start building, [read the developer documentation →](https://www.figma.com/plugin-docs/intro/)

## Use plugins with parameters

Anyone can [install and run a plugin](https://help.figma.com/hc/en-us/articles/360040450413) and use its parameters.

Users on the [Organization plan](https://help.figma.com/hc/en-us/articles/360039957374) can publish plugins with parameters to their Organization privately.

**[Learn how to run plugins →](https://help.figma.com/hc/en-us/articles/360042532714)**

## Publish plugins with parameters

### New plugins that are parameter-only

Plugins that only use parameters (those without a user interface outside of quick actions) can be published to the community and will be approved according to our [plugin review guidelines](https://help.figma.com/hc/en-us/articles/360039958914).

**[Learn more about publishing plugins to the Figma Community →](https://help.figma.com/hc/en-us/articles/360042293394)**
