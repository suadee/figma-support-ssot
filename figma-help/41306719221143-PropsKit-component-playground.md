## 내용

When you're generating plugins with the Figma agent, there are many different controls that can be included for your resource. The controls correspond to Figma's PropsKit components, which the agent uses to build the controls for your resource.

The component playground on this page shows you the different components you can add as controls to your generative plugins.

Use the playground to experiment with different controls and find what fits
best
for your resource. Then, click

**Copy prompt** and paste the prompt with your configuration
into
your chat with the Figma agent.

**Caution:** PropsKit components can only be used with generative plugins. Learn more about [generative plugins and shaders](https://help.figma.com/hc/en-us/articles/41147702210071).
