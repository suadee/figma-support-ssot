## 내용

Who can use this feature

Available on [any team or plan](https://help.figma.com/hc/en-us/articles/360040328273)

You need a Figma account to duplicate Community files

Community files are FigJam or Figma Design files that creators have shared with the Community. Files can contain libraries and UI, tools and processes, or other educational resources. Every file has a resource page where you can view file information and preview the file or prototype. [Interact with these previews](https://help.figma.com/hc/en-us/articles/360051741274) like you would any regular embed.

![iOS 16 UI Kit design file page on Figma Community, showing components and templates for download and use in iOS design projects.](https://help.figma.com/hc/article_attachments/19397589575575)

Licensing terms for Community files vary depending on if the file is free or paid. Learn more about [Figma Community copyright and licensing →](https://help.figma.com/hc/en-us/articles/360042296374)

Note: You can save your favorite files so that you can more easily access them in the future. Learn more about [saving files →](https://help.figma.com/hc/en-us/articles/360052033434-Community-follows-and-likes)

## Duplicate files

Duplicate a file to add a copy of it to your drafts folder. This duplicate is an entirely new file, so you won’t be able to access any version history, comments, or permissions from the original file.

Changes made to the original file are not reflected in duplicated files. You must duplicate the updated file to view any changes.

**Caution:** You can't duplicate Community files when browsing Community as a team or organization admin. Make sure you're browsing on your own profile before trying to duplicate a file.

1. Click **Open in Figma**, **Open in FigJam**, or **Buy** (paid files).   
   For paid Figma Design files, you can click **Get free preview** to trial the content before purchasing. Learn more about [purchasing resources →](https://help.figma.com/hc/en-us/articles/11786473291159)
2. Select the account where you want to add the file.
3. A copy of the file is added to your drafts. Every file you duplicate from the Community will have (Community) appended to the file name.

If you no longer need the file, you can [delete it from your drafts](https://help.figma.com/hc/en-us/articles/360047512294).

Note: Weave workflow templates are pre-built AI workflows available on Figma Community. When you select a Weave workflow template, you'll be redirected to Figma Weave—a separate creative platform at [weave.figma.com](https://weave.figma.com)—where you can duplicate the workflow to your Figma Weave account and start exploring.

You'll need a Figma Weave account to duplicate Weave workflow templates. If you already have a Figma account, you can sign in at [weave.figma.com](https://weave.figma.com) using your existing Figma credentials.

[Learn more about Figma Weave](https://help.weavy.ai/en/).
