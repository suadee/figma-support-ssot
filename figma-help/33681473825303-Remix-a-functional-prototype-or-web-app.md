## 내용

Before you start

Who can use this feature

Figma Make is available for [Full seats](https://help.figma.com/hc/en-us/articles/360039960434) on paid plans.

You can try Figma Make on [other seats and plans](https://help.figma.com/hc/en-us/articles/31722591905559-Figma-Make-FAQs#h_01JZQX7SQ7RX4SJ65DHDTYS2EK).

When you’re viewing functional prototypes and web apps created with Figma Make in the Figma Community, you can choose to remix them.

Remixing lets you take an existing functional prototype or web app and make changes using the AI chat, the Point to edit tool, and by directly editing the code.

**Note:** You can’t republish remixed functional prototypes or web apps, but you can still publish them to the public web or share the Figma Make file with others.

![](https://help.figma.com/hc/article_attachments/33681504548247)

To remix a functional prototype or web app:

1. Go to the Community page for the functional web app or prototype that you want to remix.
2. In the upper-right corner, click **Remix**.
3. Select the team or organization where you want to remix the Figma Make file.

After you select your team or organization, Figma Make opens. You can use the [AI chat and Point to edit](https://help.figma.com/hc/en-us/articles/31304485164695-Create-and-edit-a-functional-prototype-or-web-app#h_01JTEVBJ05NZX49SGHGNDNT9T8) to make changes to the functional prototype or web app, or [edit the code](https://help.figma.com/hc/en-us/articles/33649966245783-Edit-the-code-of-a-functional-prototype-or-web-app) directly. You can also [publish to the public web](https://help.figma.com/hc/en-us/articles/31304586129559), or [share the functional prototype or web app](https://help.figma.com/hc/en-us/articles/31304548273047-Preview-a-functional-prototype-or-web-app#h_01JTEZH2Q3T7YR8ZTPXT567DMX) with others.
