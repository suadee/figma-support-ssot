## 내용

Before you start

Who can use this feature

Currently available on
[Governance+](https://help.figma.com/hc/articles/22932461365015-Figma-for-Government)
plans.

Anyone with `Admin` access to an organization
can manage this setting.

When helping customers troubleshoot issues or potential bugs, Figma’s Support team may need to view information like the names of files, teams, and users in question to find a solution. By default, Figma Support has the ability to do this for the sole purpose of providing customer support. If an organization does not want to allow Figma’s Support team to view information about their resources, even when members of the organization have asked for assistance, an organization Admin is able to turn off Figma’s visibility at any time.

By turning off this toggle, you’ll be blocking visibility of:

- Team names
- Project names
- File names
- Usernames and account details

If you need to share specific files with Figma Support for help, you can still do so with this setting turned off.

Note: Figma can never directly access your files or information about your account without explicit and temporary permission.

To manage Figma Support’s visibility of resource information:

1. Visit the [file browser](https://help.figma.com/hc/en-us/articles/14381406380183) and select **Admin** in the left sidebar.
2. Go to **Settings**.
3. Under **Other**, toggle off **Figma Support visibility**.

This setting can always be turned off by an Admin as needed for working with customer support.

[Learn more about working with Figma Support](https://help.figma.com/hc/en-us/sections/19818052690455).
