## 내용

## Create a design file

Now that your account is up and running, you’re ready to create your first design file. Select Drafts in the sidebar and then select Design from the menu. The new file will open in the Figma Design editor.

![Cursor selecting "Design" from the file options to create a new Figma Design file.](https://help.figma.com/hc/article_attachments/33966617587607)

**Tip:** You can also create new design files by typing [figma.new](https://figma.new) in the address bar.

## Lesson wrap up

We're ready to start designing! Before moving on, make sure you have:

- Created your Figma account
- Uploaded an avatar picture
- Created your first design file

In the next lesson, we'll learn how to navigate the Figma Design editor and set our design file up for the rest of the course.
