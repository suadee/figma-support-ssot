# Import PowerPoint files to Figma Slides

Before you Start

Who can use this feature

Available on [any plan](https://help.figma.com/hc/en-us/articles/360040328273-Choose-a-Figma-Plan)

Anyone can import PowerPoint files to their drafts

Anyone with `can edit` access to a team can import PowerPoint files to that team

You can import PowerPoint files (.pptx) to Figma Slides.

## Import PowerPoint files

1. From the [file browser](https://help.figma.com/hc/en-us/articles/14381406380183), click  **Import**.
2. Click **Import from computer**.
3. Select the .pptx file you want to import and click **Open**. Depending on the file size, the import process may take a few seconds.
4. Click **Done**.

Once imported, you can modify existing slides, [add new slides](https://help.figma.com/hc/en-us/articles/24244212438935-Add-delete-and-organize-slides), and [edit presenter notes](https://help.figma.com/hc/en-us/articles/24245848829847-Add-and-view-presenter-notes).

Note: You can also drag and drop files from your computer to import them to the file browser.

## PowerPoint (.pptx) import limitations

We recommend that you review the imported slide deck carefully, as some objects may be changed or removed during the import process.

### Missing fonts

Once imported, all text layers will be updated to use the default Figma Slides font, **Inter**. You can use the [text settings](https://help.figma.com/hc/en-us/articles/360039956634-Explore-text-properties) in the right sidebar to select a new font and change the font properties.

### Unsupported objects

The following objects will not be imported to Figma Slides:

- **Videos**: You can re-upload the video using the [media picker](https://help.figma.com/hc/en-us/articles/25427361244183-Add-images-and-videos-to-a-slide).
- **Tables**: You can [recreate the table](https://help.figma.com/hc/en-us/articles/30600895164439), or take a screenshot of the original table and upload it as an image.
- **Diagrams:** You can recreate the diagram using the [shape tools](https://help.figma.com/hc/en-us/articles/30487313436183-Add-shapes-with-text-to-slides), or take a screenshot of the original diagram and upload it as an image.
- **Animations:** You can use the animation settings to add new [slide transitions](https://help.figma.com/hc/en-us/articles/24244588378007-Use-slide-transitions) or [object animations](https://help.figma.com/hc/en-us/articles/30601608159383).
