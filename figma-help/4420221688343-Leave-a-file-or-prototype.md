## 내용

Who can use this feature

Available on [any team or plan](https://help.figma.com/hc/en-us/articles/360040328273-Choose-a-Figma-Plan)

Has someone shared a file or prototype with you and you no longer want access to it? You can leave the file to remove it from your account.

- You can only leave files you have explicit access to. If you have inherited access to the file via a project or team, you need to [leave that project or team](https://help.figma.com/hc/en-us/articles/360039965653) to remove your access.
- If you're the owner of the file, you need to [transfer ownership of the file](https://help.figma.com/hc/en-us/articles/360038512093) before you can leave it.

## Leave a file

To leave a file:

1. Access the share modal in one of two ways:
   - Find the file in the file browser, then right-click on the file and select **Share**.
   - From an open file, click **Share** in the top right.
2. Find your name in the list of collaborators.
3. Click on your current access level and select **Leave** from the options.

## Leave a prototype

To leave a prototype:

1. From the prototype, click **Share prototype** in the top right.
2. Find your name in the list of collaborators.
3. Click on your current access level and select **Leave** from the options.
