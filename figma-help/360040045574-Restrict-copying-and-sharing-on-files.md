## 내용

Who can use this feature

Available on [all paid plans](https://help.figma.com/hc/en-us/articles/360040328273)

Requires `can edit` access

By default, anyone with `can view` access to a file can copy, share, and export the file and its contents.

If you restrict the file's sharing settings, people with `can view` access won't be able to:

- Copy frames, layers, or text from the canvas
- Duplicate the file to their account
- Export any assets from the file
- Save a [copy of the file to your computer](https://help.figma.com/hc/en-us/articles/8403626871063-Save-a-local-copy-of-files)

Figma will apply this restriction to everyone with `can view` access to the file, including people specifically invited to the file, or who join the file via a shared link.

**Note**: This setting only applies to individual files. There isn't a way to apply this setting to multiple files at once.

## Restrict copying, sharing, and exporting from a file

1. Click **Share** in the right sidebar.
2. Select the current sharing permissions for the file to open up the **Share settings**.
3. Under **Advanced**, deselect the **Viewers can copy, share, and export from this file** setting.
