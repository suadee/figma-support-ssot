## 내용

Who can use this feature

Available on [all paid plans](https://help.figma.com/hc/en-us/articles/360040328273)

Requires `can edit` permission to the file

Sometimes, you need to share a design file, prototype, or FigJam board with a broad audience, but want only the intended people to see it.

For example, you might want to:

- Collect usability feedback on prototypes
- Share designs with external clients
- Share with people who don’t have a Figma account

In cases like these, you can add password protection from a file’s share settings. This way, you can safely share a public link, but only people who know the password will be able to view the file.

Keep in mind:

- You can add password protection to individual Figma Design files or to select prototypes within a file. If a password is set on the design file, prototypes within the file will also require that password.
- In FigJam, password protection covers both the FigJam board and any open sessions
- It’s not possible to set a password on files published to the Figma Community
- Figma hides previews of password-protected files in embeds, link unfurls, and file thumbnails

## Add a password to a file or prototype

Everyone accessing a password-protected file needs to enter the password, unless they have `can view` or `can edit` access to the project the file is in, or they are [invited to the file](https://help.figma.com/hc/en-us/articles/360040531773.html#Send_invitations_1) using their email address.

When adding a password to a file, you can create one yourself, or use one generated by Figma.

- Passwords you create can be any length and use any character types
- Passwords generated by Figma combine four randomly-chosen words, so they’re strong and unique each time

**Note**: If you can't create your own password, your organization may [require autogenerated passwords only](https://help.figma.com/hc/en-us/articles/5726756336791).

It’s not possible to view a password after it has been set. If you forget to copy the password, you can [reset it](#h_01H674PGTZ0YQ7NZ4JMQ2G8VG4).

1. Open the file or prototype and share it.
   - For files, click **Share** in the right sidebar.
   - For prototypes, click **Share prototype** in the toolbar.
2. Click **Can view/Can edit** to change the access setting on the file or prototype.
3. Change the access to **Anyone**.
4. Under **Additional security**, check **Password required**.
5. Enter a password in the **Add password** field, or click **Generate** to generate one.
6. Click **Save**.

![Share settings panel with "Password required" checked, allowing a password-protected file share, with an option to generate a password.](https://help.figma.com/hc/article_attachments/29423309473047)

**Tip:** On the [Enterprise plan](https://help.figma.com/hc/en-us/articles/360040328273), you can also [set an expiration date on public links in design files](https://help.figma.com/hc/en-us/articles/16142157359255).

## Reset a password on a file or prototype

1. Open the file or prototype and share it.
   - For files, click **Share** in the right sidebar.
   - For prototypes, click **Share prototype** in the toolbar.
2. Click **Can view/Can edit** next to **Anyone with password** to change the access setting on the file or prototype.
3. Under **Password required**, click **Reset**.
4. Enter a password in the **Add password** field, or click to generate one.
5. Click **Save**.

**Caution:** Changes to a file’s password take effect after you click **Save**. Anyone with the file open will need to enter the new password to continue accessing the file.

## Remove password protection from a file

After removing password protection from a file, Figma immediately reloads the file for anyone with it open.

1. Open the file or prototype and share it.
   - For files, click **Share** in the right sidebar.
   - For prototypes, click **Share prototype** in the toolbar.
2. Click **Can view/Can edit** next to **Anyone with password** to change the access setting on the file or prototype.
3. Under **Additional security**, uncheck **Password required**.
