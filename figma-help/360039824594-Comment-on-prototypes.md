## 내용

Before you Start

Who can use this feature

Available on [all plans](https://help.figma.com/hc/en-us/articles/360040328273-Plans-and-teams-in-Figma)

Requires view access to the prototype

New to comments in Figma? [Get started with comments →](https://help.figma.com/hc/en-us/articles/360039825314)

Use comments in presentation view to give and respond to feedback on a prototype. You can comment if you're signed in to Figma and have at least view access to the file.

Note: Comments aren't displayed when the prototype's [toolbar and footer are hidden](https://help.figma.com/hc/en-us/articles/360040318013). Show the toolbar and footer to add or view comments. [**View prototypes →**](https://help.figma.com/hc/en-us/articles/360040318013)

## Add comments

Comments you add in a prototype also appear in the underlying design file.

1. Open the prototype in presentation view.
2. Go to the screen you want to comment on.
3. Select  in the toolbar, or press `C`. Your cursor changes to a .
4. Choose where to add the comment:
   - Select a location to pin the comment.
   - Click and drag to select a region.
5. Enter your comment. You can [mention collaborators](https://help.figma.com/hc/en-us/articles/360041068574#mention) or [add emoji](https://help.figma.com/hc/en-us/articles/360041068574#emoji).
6. Select  to send your comment.

Select  again, or press `Esc`, to exit comment mode.

**Tip:** You can also [comment on prototypes from the Figma mobile app.](https://help.figma.com/hc/en-us/articles/1500007537281)

## Reply to comments

In presentation view, select a comment to open its thread and reply.

1. Select a comment pin  to open the message.
2. Select the **Reply** field and enter your response. You can [mention collaborators](https://help.figma.com/hc/en-us/articles/360041068574#mention) or [add emoji](https://help.figma.com/hc/en-us/articles/360041068574#emoji).
3. Select  to add your reply to the thread.

## Resolve comments

When you've reached an agreement or resolved an issue, open the comment and select  **Resolve**.

Figma hides the comment from the prototype. To access it again, show resolved comments.

## Adjust your comment settings

### Show resolved comments

Show resolved comments to display them alongside new and open comments.

1. Select  to open the file in presentation view.
2. Select **Options** in the toolbar.
3. Select **Show resolved comments**.
4. Select the setting again to hide resolved comments.

Note: If the prototype has no comments, this setting is grayed out and can't be changed.

### Show only your comments

Show only your comments to hide comments from other collaborators.

1. Select  to open the file in presentation view.
2. Select **Options** in the toolbar.
3. Select **Show only your comments**.
4. Select the setting again to show comments from all collaborators.
