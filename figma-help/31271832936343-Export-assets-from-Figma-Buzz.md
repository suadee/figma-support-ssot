## 내용

Figma Buzz was released in [open beta](https://help.figma.com/hc/en-us/articles/4406787442711) at Config 2025. It is currently available on all plans and seat types. [Learn more about what’s included in the beta.](https://help.figma.com/hc/en-us/articles/4406787442711)

Who can use this feature

While in beta, Figma Buzz is available on all seats and plans.

Video import and export is available on paid plans.

You can export assets from Figma Buzz to share or use them outside of your file.

Depending on your organization’s approval settings, [assets may need to be approved](https://help.figma.com/hc/en-us/articles/39288847137815) before they can be exported.

## Export assets

You can export individual assets or multiple assets at once from any Figma Buzz file. Assets can be exported as static images (PNG, JPG, PDF) or MP4 videos—depending on their media type.

1. Open a Figma Buzz file.
2. Select the asset(s) you want to export. If no assets are selected, all assets in the file will be included in the export.
3. Click **Export**.
4. Use the Export modal to configure your export settings. If you’re exporting multiple assets, you can select each one in the sidebar to customize their settings individually.
   - **File type:** Options include PNG, JPG, PDF (for static images), or MP4 Video.
   - **Size**: Select a scale for your image assets. Choose 1x if you want to export assets from templates sized for certain channels in their required dimensions.
   - **Color profile (PDF only):** Choose to export content using sRGB or Display P3. Learn more about [color profiles](https://help.figma.com/hc/en-us/articles/360039825114-Manage-color-profiles-in-design-files).
5. Click **Export assets**.
   - If you're using Figma Buzz in the browser, files will download to your browser’s default location.
   - If you’re using the Figma desktop app, you’ll be prompted to rename the file and choose a destination.

If you’re using Figma Buzz in the browser, the exported file will be sent to the download location set in your browser’s preferences. If you’re using the Figma desktop app, you are prompted to rename the file and choose where to send the export.

## Export assets from Figma Design

While in design mode, you can access the same [export formats and settings](https://help.figma.com/hc/en-us/articles/13402894554519-Export-formats-and-settings) available in Figma Design. This allows you to export content to PNG, JPG, SVG, and PDF, and gives you more control over the exported file. You can export entire campaigns or individual assets within Buzz.

1. Select the assets you want to export. To select an entire campaign to export, do one of the following:
   - **In asset view:** Click a blank space on the canvas to deselect everything
   - **In grid view:** Hover just above an asset and click the blue outline that appears
2. Click  plus in the **Export** section of the right sidebar to add an export configuration. You can add as many export configurations to a selection as needed.
3. Configure the export settings. Learn more about [Figma's export formats and settings](https://help.figma.com/hc/en-us/articles/13402894554519).
4. (Optional) Click **Preview** to see how your asset will look. If you have multiple layers selected, the **Preview** setting does not display.
5. Click **Export**.

If you’re using Figma Buzz in the browser, the exported file will be sent to the download location set in your browser’s preferences. If you’re using the Figma desktop app, you are prompted to rename the file and choose where to send the export.
