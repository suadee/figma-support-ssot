Before you Start

Who can use this feature

Available on [any plan](https://help.figma.com/hc/en-us/articles/360040328273-Choose-a-Figma-Plan)

Anyone with `can edit` access to a file can use Figma Draw features

Add personality and artistic flare to your designs with Figma Draw. Figma Draw lets you access illustration tools like [brushes](https://help.figma.com/hc/en-us/articles/31440438150935), [transforms](https://help.figma.com/hc/en-us/articles/31440427042839), and [textures](https://help.figma.com/hc/en-us/articles/360041488473-Apply-shadow-or-blur-effects), directly inside a Figma Design file.

## Enter Figma Draw

Figma Draw is a set of visual design tools within the Figma Design editor. To open Figma Draw, click  **Draw** in the toolbar.

![](https://help.figma.com/hc/article_attachments/40102330562583)

**Note:** Some Figma Design features are not available while in Figma Draw. Switch the toolbar toggle to  **Design** to access all Figma Design features.

## Navigate Figma Draw

While using Figma Draw, the toolbar and sidebars change to provide a set of illustration-focused tools and properties.

### The toolbar

![imageTemplate-1.png](https://help.figma.com/hc/article_attachments/40102330563991)

The toolbar changes to give you quick access to the [Pen](https://help.figma.com/hc/en-us/articles/360040450213-Vector-networks), [Brush](https://help.figma.com/hc/en-us/articles/31440438150935), and [Pencil](https://help.figma.com/hc/en-us/articles/31440438150935) tools.

### The right sidebar

![imageTemplate-2.png](https://help.figma.com/hc/article_attachments/40102284578583)

The right sidebar changes to a more streamlined view of illustration-related properties, with slider controls that help you balance artistic feel with pixel-perfect adjustments. You can also access [transforms](https://help.figma.com/hc/en-us/articles/31440427042839-Create-patterns-with-transforms) from the right sidebar while in Figma Draw.

### The left sidebar

![imageTemplate.png](https://help.figma.com/hc/article_attachments/40102330565911)

The **Layers** section in the left sidebar displays an enlarged visual preview of the layer’s contents. Double-click on the layer preview to zoom directly to that layer on the canvas.
