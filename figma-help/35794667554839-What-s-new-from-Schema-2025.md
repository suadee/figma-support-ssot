## 내용

Schema is Figma’s design systems conference. In this article, we’ll recap everything that was announced at Schema 2025. Some of these features are not generally available yet, so we’ll indicate their availability and update this article if there are any changes.

To learn more about our vision for design systems and everything we announced at Schema, watch the Schema 2025 keynote or check out our [blog](https://www.figma.com/blog/schema-2025-design-systems-recap/).

## Figma MCP server GA Available now

Before you Start

Who can use this feature

The remote server is available on [all seats and plans](https://help.figma.com/hc/en-us/articles/360040328273-Figma-plans-and-features)

The desktop server is available on a [Dev or Full seat](https://help.figma.com/hc/en-us/articles/360039960434-Manage-seats-in-Figma) for [all paid plans](https://help.figma.com/hc/en-us/articles/360040328273-Figma-plans-and-features)

You must use a code editor or application that supports MCP servers, such as VS Code, Cursor, Windsurf, Claude Code

Users on the Starter plan or with View or Collab seats on paid plans will be limited to up to 6 tool calls per month.

Users with a Dev or Full seat on the Professional, Organization, or Enterprise plans have per minute rate limits, which follow the same limits as the Tier 1 [Figma REST API](https://developers.figma.com/docs/rest-api/rate-limits/). As with Figma’s REST API, Figma reserves the right to change rate limits.

The Figma MCP server is now generally available to all users. Use the Figma MCP server to deliver Figma context to your code editor and improve your AI generation workflow.

**Want to learn more about the Figma MCP server?** Check out the full [Figma MCP collection](https://help.figma.com/hc/en-us/sections/35280374295831-Figma-MCP-collection), or get started with the [Figma MCP installation guide](https://developers.figma.com/docs/figma-mcp-server).

## Code Connect UI GA Available now

Before you Start

Who can use this feature

 

Code Connect UI is available on [Organization and Enterprise plans](https://help.figma.com/hc/en-us/articles/360040328273-Figma-plans-and-features)

Requires a [Full or Dev seat](https://help.figma.com/hc/en-us/articles/360039960434)

The [Code Connect UI](https://developers.figma.com/docs/code-connect/code-connect-ui-setup/) is now generally available to users on the Organization and Enterprise plans. The new UI makes it easier to directly connect Figma with your design system codebase for faster setup and less maintenance over time.

As part of this release, we’ve also added new features for users that have connected their GitHub repositories to the UI:

- **Component mapping suggestions:** Get recommendations for which code files to connect to your Figma components
- **AI-generated code snippets:** See example code for the connected components based on your actual source files
- **MCP usage instructions:** Provide guidelines for how that component should be used by LLMs as part of Figma MCP server responses

![](https://help.figma.com/hc/article_attachments/35840299891479)

### Code Connect UI FAQ

**Do I need to connect to GitHub to use Code Connect UI?**

Connecting to GitHub makes it easier to set up Code Connect UI, but you can still create connections using manually provided links or by using Code Connect CLI.

**How is Code Connect related to Figma’s MCP server?**

Connections made in Code Connect are accessible through Figma’s MCP server and provided as additional design system context for AI-enabled code editors or applications.

## Variable mode limit increase Available now

We’ve increased the variable mode limit for [Professional and Organization](https://help.figma.com/hc/en-us/articles/360040328273-Figma-plans-and-features) plan users:

- **Professional plan:** Up to 10 variable modes per collection
- **Organization plan:** Up to 20 variable modes per collection

## Extended collections Available now

Before you Start

Who can use this feature

 

Extended collections will be available to users with [Full seats](https://help.figma.com/hc/en-us/articles/360039960434-Manage-seats-in-Figma#editor) on the [Enterprise plan](https://help.figma.com/hc/en-us/articles/360040328273-Figma-plans-and-features)

Extended collections are a new way to manage multi-brand design systems. Design systems authors can now publish a core design system that designers across their company can extend by adding their own brand-specific themes.

An extended collection is tied to its parent design system and will inherit any changes, such as new variables or color updates. This allows the extended collection to stay up-to-date with the parent system, while still maintaining brand colors and values that have been explicitly overridden. Learn more about [extended collections](https://help.figma.com/hc/en-us/articles/36346281624471-Extend-a-variable-collection).

## Import npm packages into Figma Make Available now

Before you Start

Who can use this feature

 

Public npm packages are available on [any plan](https://help.figma.com/hc/en-us/articles/360040328273-Figma-plans-and-features)

Private npm packages are available on the [any paid plan](https://help.figma.com/hc/en-us/articles/360040328273-Figma-plans-and-features)

You can bring your production-ready React design system into Figma Make to build high-fidelity prototypes that use the exact same code as your production apps. This lets you prototype, test, and ship using one consistent source of truth. Learn more about [bringing your package design system into Figma Make](https://help.figma.com/hc/en-us/articles/35946832653975-Use-your-design-system-package-in-Figma-Make#h_01K8PAEVXB28V7K2AHHW7CBBQF).

## Slots Available now

Before you Start

Who can use this feature

Slots is currently in beta and is gradually rolling out to everyone. It will be available on [all plans](https://help.figma.com/hc/en-us/articles/360040328273-Figma-plans-and-features). Requires a [Full seat](https://help.figma.com/hc/en-us/articles/360039960434-Manage-seats-in-Figma#editor) on paid plans.

Slots are placeholder containers inside components that let you add, edit, and customize component instances without needing to detach. They allow for more flexibility in design and consistency with code. You can also specify which instances a slot accepts, ensuring compliance with your design system. Learn more about [slots](https://help.figma.com/hc/articles/38231200344599/).

![](https://help.figma.com/hc/article_attachments/35810181962007)

## Make kits Available now

Before you Start

Who can use this feature

 

Make kits are available on [all paid plans](https://help.figma.com/hc/en-us/articles/360040328273-Figma-plans-and-features).

Make kits let you import Figma Design libraries into Figma Make, so you can generate high-quality prototypes, React code components, and a CSS file from your components, styles, and variables. Learn more about [Make kits](https://help.figma.com/hc/en-us/articles/39241689698839).

## Check designs Early access sign-up

Before you Start

Who can use this feature

Check designs is currently only available to people in the early access program. It will be available to [Organization and Enterprise plan](https://help.figma.com/hc/en-us/articles/360040328273-Figma-plans-and-features) users.

You can apply to join the early access waitlist [here](https://docs.google.com/forms/d/e/1FAIpQLSeaZNBEWTahxSYh4FDpUNzQlwM3HKw_Eg0U-hwgXETCtAxm8A/viewform). Please note that applying does not guarantee access.

Check designs lets you audit and align designs to your design system before you hand them off to developers. Check designs will surface elements using hard-coded values and suggest the right variable to use in that context. Once audited, you can hand off those designs with confidence.

![](https://help.figma.com/hc/article_attachments/35939835050647)

### Check designs FAQ

How do I get access to check designs?  
You can apply to join the early access waitlist [here](https://docs.google.com/forms/d/e/1FAIpQLSeaZNBEWTahxSYh4FDpUNzQlwM3HKw_Eg0U-hwgXETCtAxm8A/viewform?usp=header). Please note that applying does not guarantee access.  
  
How will I know if I’ve been added to the early access program?  
You’ll get an email from the Figma team letting you know once you have access to the early access program.  
  
I’m an admin. Can I request that my whole team or organization be given early access to check designs?  
Yes, your entire team or organization will be given access to the check designs early access program.
