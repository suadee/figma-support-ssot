## 내용

Before you start

Who can use this feature

Available on [all plans](https://help.figma.com/hc/en-us/articles/360040328273).

You need `can edit` access to a team to create a project in that team

Projects allow you to group related files together. Projects are created within [teams](https://help.figma.com/hc/en-us/articles/360039480614-Manage-a-Starter-or-Professional-team).

On the Professional, Organization, or Enterprise plan, you can create unlimited projects. On the Starter plan, you can create one project.

![A graphic showing the hierarchy of resources in Figma: files sit in projects, which sit in teams, which sit in workspaces (enterprise only), which sit in an organization (organization and enterprise only).](https://help.figma.com/hc/article_attachments/35463257718935)

## Create a new project

To create a new project:

1. Open the [file browser](https://help.figma.com/hc/en-us/articles/14381406380183-Guide-to-the-file-browser) by logging in to your account at [figma.com](https://figma.com/).
   - If you’re inside a file, click the  Figma menu > **Back to files.**
2. View a list of projects within a team.
   - On the Starter or Professional plan, select  **All projects** from the left sidebar.
   - On the Organization or Enterprise plan, select  **All teams** from the left sidebar. Then, select a team.
   - On the Enterprise plan with workspaces enabled, select  **All workspaces** from the left sidebar. Then, select a team from the **Your teams** tab.
3. Click **+ Project.**
4. Enter in a project name.
   1. On the Organization or Enterprise plans, you’ll also set [organization access to the project](https://help.figma.com/hc/en-us/articles/35361119554711-File-and-project-permissions).
5. Click **Continue**.
6. Enter in the email addresses of any users or [user groups](https://help.figma.com/hc/en-us/articles/38020006913943) you would like to directly invite to the project, and set their [project permissions.](https://help.figma.com/hc/en-us/articles/35361119554711-File-and-project-permissions)
7. Click **Create project**.

## Invite to a project

You can invite users to an existing project at any time.

- If you have `can edit` permissions on a project, you can invite other users to the project with `can edit` or `can view` permissions.
- If you have `can view` permissions on a project, you can only invite others with `can view` permissions.

To invite others to a project:

1. Open the [file browser](https://help.figma.com/hc/en-us/articles/14381406380183-Guide-to-the-file-browser) by logging in to your account at [figma.com](https://figma.com/).
   - If you’re inside a file, click the  Figma menu > **Back to files.**
2. View a list of projects within a team.
   - On the Starter or Professional plan, select  **All projects** from the left sidebar.
   - On the Organization or Enterprise plan, select  **All teams** from the left sidebar. Then, select a team.
   - On the Enterprise plan with workspaces enabled, select  **All workspaces** from the left sidebar. Then, select a team from the **Your teams** tab.
3. Select a project.
4. Click **Share**.
5. In the Share modal, enter an email address or [user group](https://help.figma.com/hc/en-us/articles/38020006913943-Create-and-manage-user-groups) name in the invite field, then set their [project permissions](https://help.figma.com/hc/en-us/articles/35361119554711-File-and-project-permissions).
6. Click **Send invite**.
