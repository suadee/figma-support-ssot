## 내용

We are aware of an issue impacting Illustrator 29.8.1 that prevents copying and pasting vectors into Figma. As a workaround, we recommend [exporting your artwork as an SVG](https://helpx.adobe.com/illustrator/using/exporting-artwork.html), then [importing it into Figma](https://help.figma.com/hc/en-us/articles/360040028034-Add-images-and-videos-to-designs).

Starting with Adobe Illustrator 29.7.1, SVG code is no longer included by default when copying vectors. This causes vectors to be converted into static images when pasted into Figma.

To prevent this issue, enable **Include SVG Code** in your Adobe Illustrator preferences. Learn more about [managing Adobe Illustrator preferences](https://helpx.adobe.com/illustrator/using/setting-preferences.html).
