## 내용

Who can use this feature

Available on [all plans](https://help.figma.com/hc/en-us/articles/360040328273)

Admins only

### Change a team name

Starter or Professional plan Organization or Enterprise plan

1. From the [file browser](https://help.figma.com/hc/en-us/articles/14381406380183), click  **All projects**.
2. Click the  **down arrow** icon next to the team name and select **Rename** from the menu that appears.

1. From the file browser, navigate to the team you'd like to rename.
2. Click the  **down arrow** icon next to the team name and select **Rename** from the menu that appears.

**Tip:** You can also rename the team from the sidebar in the file browser. Right-click the team name and choose **Rename** from the menu.

### Change a team icon

You can customize a team by replacing the default icon with your own image.

Starter or Professional plan Organization or Enterprise plan

1. From the [file browser](https://help.figma.com/hc/en-us/articles/14381406380183), click  **All projects**.
2. Click the  **down arrow** icon next to the team name and select **Change icon.**
3. Upload an image.

1. From the file browser, navigate to the team you'd like to update.
2. Click the  **down arrow** icon next to the team name and select **Change icon.**
3. Upload an image.

**Note**: The minimum size for a team icon is 500px by 500px.
