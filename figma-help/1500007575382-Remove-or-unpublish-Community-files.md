## 내용

Who can use this feature?

Supported on any team or plan.

Applies to both Figma Design and FigJam files.

You must have can edit access to the file to publish or unpublish files.

Which profile(s) you can publish to depends on your permissions within a team or organization. **[Learn who can publish files and plugins →](https://help.figma.com/hc/en-us/articles/360041423614)**

Choose whether to remove yourself as a creator — to remove it from your profile without removing the file— or unpublish the file to prevent others from duplicating it.

You need to browse as an admin to manage files or plugins published to a team or organization profile. [**Manage team and organization profiles →**](https://help.figma.com/hc/en-us/articles/4404108672663)

## Unpublish a file

Unpublish the file to remove the file listing from the Community. This will remove the file from your Community profile and the profiles of any other creators.

Note: Paid files cannot be unpublished but can be delisted. In most cases, delisted files are still accessible by users who purchased them. Learn more about [delisted paid resources](https://help.figma.com/hc/en-us/articles/12843697930519) →

Removing or unpublishing a file doesn't affect duplicates of the file. Anyone who duplicated the file before it was removed can continue to access and edit their copy of the file.

You can remove a published file from the original file, or from the file page in the Community.

Note: Only an organization admin can unpublish a file or plugin on an organization's profile. If you're not an organization admin, you can still choose to [remove the file from your profile](#remove-creator).

- A 

  ### Original file

  1. Open the file in the editor.
  2. Click the **Share** button in the toolbar.
  3. Select the **Publish to Community** tab at the top of the modal.
  4. Click the **Unpublish** button to remove the file from the Community, and any creator's profile(s).
- B 

  ### File listing

  1. Open the file page in the Figma Community.
  2. Click the **Unpublish** button at the bottom of the page.
  3. Click **Unpublish** in the dialog to confirm.

## Remove yourself as a creator

If you're listed as a creator of a resource, but don't want to have it on your Community profile, you can remove yourself as a creator.

This will remove you as a creator on the file listing and from your own profile, but won't remove the file from the Community.

Note: If you are the designated payee of a paid file, you cannot remove yourself as the creator.

Anyone who has edit access to the file can add and remove creators.

1. Open the file page in the Figma Community.
2. Click the **Remove yourself from creators** button at the bottom of the page.
3. Click **Remove** in the dialog to confirm.
