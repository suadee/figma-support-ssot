## Common errors and solutions

|  |  |
| --- | --- |
| **Error** | **Solution** |
| Something went wrong | Reload tab and try again |
| Couldn’t connect | [Check your network settings](https://help.figma.com/hc/en-us/articles/23347716725527) |
| Wait a bit before trying again | Your Figma account may be meeting a rate limit |
| Try again, with a smaller selection | Try running action with a smaller selection |
| AI features unavailable | Figma is experiencing a connection issue |
| Blocked—try different input | Your prompt may violate usage terms |
| Reload needed | Reload your tab |
| Can’t use again until [Date] | Rate limit reached |
| No layers need naming | No layers in selection have default names |
| Missing a font—couldn’t rewrite | [Resolve missing font issues](https://help.figma.com/hc/en-us/articles/360039956994) |
| Couldn’t do that—try adjusting fonts or colors | Try a new prompt |
| Try a smaller selection | Break up selection into smaller sections |
| Try again, with a bit less text | Try again, with a bit less text |
| First, give these layers names | Name layers to be used in prototypes |
