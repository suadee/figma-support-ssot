## 내용

Before you start

Who can use this feature

Available on [any paid plan](https://help.figma.com/hc/en-us/articles/360040328273-Choose-a-Figma-Plan)

Anyone with `can edit` access to a file can add libraries

Add [Figma Design libraries](https://help.figma.com/hc/en-us/articles/360041051154-Guide-to-libraries-in-Figma) to your Figma Buzz files to access your team’s components, variables, and styles while creating assets. Using library assets helps ensure your assets are always in sync with your company's colors, fonts, and logos.

Note: Adding a library to your Figma Buzz file lets you insert components, styles, and variables directly into your asset. If you’re creating templates that use components or variables, see [Build configurable Figma Buzz templates](https://help.figma.com/hc/en-us/articles/35659298120215) with your design system to learn how these features work when published as templates.

## Add a library to your file

Admins can approve or [enable libraries for your team](https://help.figma.com/hc/en-us/articles/360039234953-Enable-a-library-for-a-team). You can search for and add any libraries that have been made available to you.

To add a library to your Figma Buzz file:

1. Click  **Inserts** from the left navigation bar.
2. Click **Add library**.
3. Search for the library name or browse available options.
4. Hover over a library and click **Add to file**.

![Add a library to a Figma Buzz file.](https://help.figma.com/hc/article_attachments/33799337727127)

## **Access library assets**

After you add a library, you can use its components, variables, and styles in your Figma Buzz file.

**Note:** If you’re using a template with guidelines, you won’t be able to use libraries unless you [remove editing restrictions](https://help.figma.com/hc/en-us/articles/31271689852823-Find-and-use-Figma-Buzz-templates#remove-guidelines) first.

### **Access components and variants**

To add a component:

1. Click  **Inserts** in the left navigation bar.
2. Choose from recently used components or select a library.
3. Drag a component onto the file.

To configure a component you’ve added:

1. Select the component.
2. In the inline toolbar, click  **Swap content**. In the modal, you can:
   - Switch between available variants
   - Swap nested instances
   - Configure component properties on nested instances

![From the swap content panel, you can swap between instances.](https://help.figma.com/hc/article_attachments/35833777850135)

### Apply styles and variables

If your library includes styles, you can apply them to text or objects in Figma Buzz.

**Text styles**

1. Select a text layer.
2. In the inline toolbar, open the font dropdown.
3. Click  **Apply styles** to view a list of available library text styles.

**Color styles and variables**

1. Select an object with a color fill.
2. In the inline toolbar, open the **Fill** dropdown.
3. From the **Color** tab, select  **Expand**.
4. Click  **Apply styles and variables** to view a list of available library color styles and variables.

![Apply styles to color fills in Figma Buzz.](https://help.figma.com/hc/article_attachments/33799322772503)

**Variable modes**

Some libraries include variable modes, which let you quickly switch between preset themes or customizations—such as colors, languages, or other variations.

To switch variable modes:

1. Select the asset on the canvas.
2. From the inline toolbar, select  **Apply variable modes**.
3. Select the variable modes you’d like to apply from the dropdown menus.

## Update a library

When a library component, style, or variable is updated, Figma notifies any file using those assets. To reflect the changes in your Figma Buzz file, you'll need to accept the updates. Learn more about [accepting library updates](https://help.figma.com/hc/en-us/articles/360039234193-Review-and-accept-library-updates).

To review and accept updates:

1. Click  **Review design updates** from the left navigation bar.
2. In the **Manage libraries** modal, click **Update** to accept an individual update, or **Update all** to accept all library updates.

![Review and accept library updates in Figma Buzz](https://help.figma.com/hc/article_attachments/33799322773911)

**Note**: If you don’t see the  **Review design updates** icon, there are no library updates to accept.

## Remove a library

If you no longer need to access a library, you can remove it from your Figma Buzz file.

1. Click  **Inserts** from the left navigation bar.
2. Click **Add library** to open the **Manage libraries** modal.
3. Click **Remove** next to the library you want to remove.

The library will no longer appear in your available libraries list. Existing components already added to your file will remain.
