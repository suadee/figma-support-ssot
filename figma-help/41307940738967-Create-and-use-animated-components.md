Anyone with `can edit` access to a file can create components

You can turn animations created in Figma Motion into reusable components and publish them to your design system. Learn more about [Figma Motion](https://help.figma.com/hc/en-us/articles/41274629073303).

## Create animated components

You create animated component the same way you’d create any standard component. Learn more about [components](https://help.figma.com/hc/en-us/articles/360038662654-Guide-to-components-in-Figma).

**Note:** Components can currently only be created in  **Design** mode.

To turn an animation into a main component:

1. Switch the toolbar toggle to  **Design**.
2. Select the layers you want to include in the component.
3. Do one of the following:
   - Click  **Create component** in the right sidebar
   - Right-click on your selection and choose **Create component**
   - Use the shortcut:
     - **Mac:** `Option` `Command` `K`
     - **Windows:** `Ctrl` `Alt` `K`

Figma will nest the layers within a special component frame. Identify components in the **Layers** panel using the purple  Component icon. You can use the [Component configuration settings](https://help.figma.com/hc/en-us/articles/7938814091287-Add-descriptions-to-styles-components-and-variables#components) to add a description and documentation link for collaborators.

Once created, you can publish animated components to a library to build your motion system. Learn more about [libraries](https://help.figma.com/hc/en-us/articles/360041051154-Guide-to-libraries-in-Figma).

## Use animated component instances

Animated component instances behave like regular instances, except that you can also view them on the Figma Motion timeline.

Animated instances appear as purple layer tracks on the timeline. You can drag to reposition them on the timeline, but the animation’s duration and properties can only be edited at the main component level. Learn more about [creating and inserting components instances](https://help.figma.com/hc/en-us/articles/360039150173-Create-and-insert-component-instances).

**Note:** Component properties cannot be modified while in Figma Motion. Switch the toolbar toggle to  **Design** to modify an instance’s component properties.
