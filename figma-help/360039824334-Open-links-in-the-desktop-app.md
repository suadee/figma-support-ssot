## 내용

Who can use this feature

Supported on [any plan](https://help.figma.com/hc/en-us/articles/360040328273-Choose-a-Figma-Plan)

**Mac:** Must be running **macOS 11**or later

**Windows:** Must be running **Windows 10** or later in a **64-bit** environment

[View minimum browser requirements for Figma](https://help.figma.com/hc/en-us/articles/360039827194).

Every Figma file has a unique URL. When you click a file link, Figma opens the file in the browser by default. Once a file is open in the browser, you can choose to open it in the [Figma Desktop app](https://help.figma.com/hc/en-us/articles/5601429983767) instead. You can also update your preferences to [always open links in the desktop app](#h_01HFCXFHYWEK5QHG1RDKYRPWN8) by default.

**Note:** The option to **Open links in desktop app** applies to individual Figma files, not other parts of the Figma application like the file browser or Community pages.

## Open the current file in the desktop app

If you have a file open in the browser, you can choose to open it in the desktop app. You must have the desktop app installed and be logged in using the same account you’re using in the browser.

1. Click  **Main menu**.
2. Select **Open in desktop app** from the dropdown.

## Open links in the desktop app by default

You can update your preferences to open file links in the desktop app by default.

1. In a Figma file, navigate to  **Main menu**> **Preferences**.
2. Select **Open links in desktop app**. If you don't see this option, check out [tips on troubleshooting Figma links](https://help.figma.com/hc/en-us/articles/22850791655831).

A checkmark appears to show the setting is enabled.

**Tip:** You can also enable this setting using the actions menu.

1. Use the keyboard shortcut to access the actions menu:
   - **Mac:** `Command` `K`
   - **Windows:** `Control` `K`
2. Type **Open links in desktop app**.
3. Check the box to toggle the setting on.

[Learn more about using the quick actions bar](https://help.figma.com/hc/en-us/articles/360040328653).

## Disable opening links in the Figma desktop app by default

To turn this setting off:

1. Open a Figma file. If you're logged into multiple accounts in the desktop app, first [select the account you want to disable this feature for](https://help.figma.com/hc/en-us/articles/1500005165741-Switch-between-accounts).
2. Select **Preferences** > **Open links in desktop app** to uncheck and disable the setting.

You can also do this using the [quick actions bar](https://help.figma.com/hc/en-us/articles/360040328653).
