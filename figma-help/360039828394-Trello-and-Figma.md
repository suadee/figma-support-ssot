## 내용

The Figma Power-up was deprecated in April 2024. Trello now natively supports the ability to embed live Figma files using [Atlassian's Smart Links →](https://community.atlassian.com/t5/Work-Management/What-are-Smart-Links-and-how-to-use-them/ba-p/2353552)

To add Smart Links to a Trello card, you can copy and paste a link to a Figma file directly into the Description section or in a comment. Files that were previously added using the Figma Power-up now appear in the **Attachments** section of the Trello card.
