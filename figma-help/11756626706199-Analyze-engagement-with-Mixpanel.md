## 내용

Mixpanel is an intuitive interface that helps you quickly analyze user engagement, conversion, and retention, and make collaborative, data-driven decisions.

You can use Mixpanel to:

- Answer questions about product usage without the need for SQL
- Measure average user retention, for example, "what % of new users return within 1 week?"
- Pinpoint where users drop off in critical flows, for example, onboarding completion
- Segment users to dig deeper into key behaviors, for example, “how often do power users adopt this new feature?”
- Build stories around data by adding context alongside metrics

Try out the Mixpanel integration [here](https://www.figma.com/community/file/1198418384409862561) using our Weekly Product Check-In template to collaborate around data with your team.

**Note:** Support for Safari is currently in progress.

# **Embed Mixpanel in a FigJam file**

Embedding a Mixpanel preview in FigJam allows you to bring data into your whiteboarding sessions. File previews are automatically synced with Mixpanel so the latest data changes in Mixpanel will be reflected in FigJam.

## Copy link from Mixpanel

## A video of the Mixpanel UI showing the steps to copy the link from Mixpanel.

1. Open the Mixpanel board or report you want to embed in a FigJam file.
2. Copy the URL or click the link icon in the top-right corner.

**Note:** To give someone access to Mixpanel, invite them to your organization / project by following [these instructions](https://help.mixpanel.com/hc/en-us/articles/360044299311-Sharing-and-Permissions). Mixpanel is not paid by seat so adding users to your Mixpanel plan will not increase your cost.

## Embed Mixpanel in FigJam

![An animation showing someone embedding a Mixpanel preview into a FigJam board.](https://help.figma.com/hc/article_attachments/11880355818775)

1. In a FigJam file, navigate to where you’d like to embed your Mixpanel preview.
2. Paste the Mixpanel URL link. Figma may ask you to authenticate with Mixpanel to access the file.
3. Now, anyone who has access to the report in Mixpanel will be able to see it in FigJam.

**Note:** While you can have as many Mixpanel embeds you want in a FigJam file, only one can be live at any given time. This is a global FigJam setting that applies to any embed to make sure FigJam continues to run smoothly.
