## 내용

Who can use this feature

Available on the [Enterprise plan](https://help.figma.com/hc/en-us/articles/360040328273)

Workspace admins can customize workspaces they manage

Organization admins can customize any workspace

![A screenshot of the All workspaces view showing the user's workspace called 'Brand' featured prominently with a colorful background, custom icon, and description that says:'brand and marketing design teams'.](https://help.figma.com/hc/article_attachments/18313522143511)

You can customize each workspace in your organization in the following ways:

- [Change the icon](#h_01HCX1WK7WX9YAS3NDJGCKRTHR)
- [Add a background color](#h_01HCX1WK7WPWZDPVGP52WXE726)
- [Add a description](#h_01HCX1WK7WC289W9CWNVVPRXTH)

### Change the icon

1. From the [file browser](https://help.figma.com/hc/en-us/articles/14381406380183), click into the workspace you'd like to update.
2. Click the  **down arrow** next to the workspace name and select **Change icon.**
3. Upload an image.

**Note**: The minimum size for a workspace icon is 500px by 500px. You can also use icons with a transparent background.

### Add a background color

The background color is visible for everyone browsing the workspace. It also appears on the featured workspace tile in  **All workspaces** for people who are members of the workspace**.**

1. From the [file browser](https://help.figma.com/hc/en-us/articles/14381406380183), click into the workspace you'd like to update.
2. Click the  **down arrow** next to the workspace name and select **Change workspace color.**
3. Click **No color** and select **Solid color** from the dropdown menu.
4. Choose one of the pre-defined options, or click the color wheel to select a custom color or enter a HEX value.

**Tip**: To remove a background color, switch **Solid color** back to **No color.**

### Add a description

Adding a description helps people understand what kind of teams, projects, and files are in the workspace. The description appears on the workspace tile in the  **All workspaces** view of your organization.

1. From the [file browser](https://help.figma.com/hc/en-us/articles/14381406380183), click into the workspace you'd like to update.
2. Click the  **down arrow** next to the workspace name and select **Edit description.**
3. Write a short description and click **Save**.
