## 내용

The Figma product suite is available in multiple language: English, Japanese, Korean, French, German, Spanish (Spain and Latin America), and Portuguese (Brazil). [You can change your Figma language preference in your account.](https://help.figma.com/hc/en-us/articles/6956360971415)

If you want to use Figma in a different language, you can use the Google Chrome browser to translate interface text to another preferred language. Machine translation is an imperfect process, so keep in mind that some text may be translated incorrectly or not translated at all. [Change Chrome languages and translate webpages.](https://support.google.com/chrome/answer/173424)

**Note**: Chrome’s translation feature is only available in the Google Chrome browser. It is not available in other browsers or when using the Figma desktop app.

You can only access the translation settings from the toolbar in Figma. This is because Figma overrides the browser’s right-click behavior in other areas to display its own menus.

## Translate using Chrome

![System cursor clicking in the top center of the Figma Design editor's left sidebar.](https://help.figma.com/hc/article_attachments/31411782720535)

1. Right-click on the Figma toolbar and select **Translate to [language]**. The language displayed may vary.
2. If the desired language is not displayed, click the translate icon on the right side of the address bar to expand it.
3. Click the triple dot menu and select **Choose Another Language**.
4. Select a language from the list of available languages.
5. Click **Translate**.

Note: Chrome won’t translate the entire page at once. You may notice a delay when translating new elements, such as opening or expanding a menu.
