## 내용

Before you Start

Who can use this feature

 

Available on [any plan](https://help.figma.com/hc/en-us/articles/360040328273-Choose-a-Figma-Plan)

Only the owner of the Figma Design file can copy content to Figma Slides using this method

You can copy over existing slide deck presentations from Figma Design to Figma Slides. This gives you to access additional presentation tools like [presenter notes](https://help.figma.com/hc/en-us/articles/24245848829847-Add-and-view-presenter-notes), [slide transitions](https://help.figma.com/hc/en-us/articles/24244588378007-Use-slide-transitions), and [live interactions](https://help.figma.com/hc/en-us/articles/24246820870807-Add-live-interactions-to-slides).

**Note:** The **Copy to Figma Slides** option appears when you select a top-level frame with an aspect ratio between 4:3 and 16:9, such as 1920 x 1080 or 1024 x 768. If the selected frame is outside this range, or if it is nested inside another frame, the **Copy to Figma Slides** option will not display. If needed, you can manually [copy and paste content](https://help.figma.com/hc/en-us/articles/4409078832791-Copy-and-paste-objects#h_01J92HBF9AD0R0NRKPMHT8NJ06) between Figma Design and Figma Slides.

![Copy to Figma Slides option in the right sidebar](https://help.figma.com/hc/article_attachments/28604685238935)

Tip: Use the Presentation [frame presets](https://help.figma.com/hc/en-us/articles/360041539473-Frames-in-Figma-Design#h_01J1981B798YQH44C63XPQRH0R) to automatically add frames with the correct dimensions.

## Copy a slide deck to Figma Slides

1. Open the Figma Design file that contains your existing slide deck.
2. Select the the top-level frames you want to copy to Figma Slides.
3. Click **Copy to Figma Slides** in the **Design** tab of the right sidebar.

A new Figma Slides deck will open in a new tab.

Note: Prototype connections between frames won’t transfer to Figma Slides, but you can use [slide transitions](https://help.figma.com/hc/en-us/articles/24244588378007-Use-slide-transitions) to add visual effects that play as you progress through your deck.
