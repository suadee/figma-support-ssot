## 내용

Before you start

Who can use this feature

Available on the Organization and Enterprise plans

Organization admins only

The Figma Community is a space where people, teams, and organizations can share and discover files, templates, plugins, and widgets.

Admins on an [Organization or Enterprise plan](https://help.figma.com/hc/en-us/articles/360040328273-Figma-plans-and-features) have the ability to manage access to Community resources from the file browser. This setting can be useful for admins to ensure that their team is using only brand-approved resources.

Access to Community resources from the **Community** page will be turned on by default. If an admin turns off this setting, access will be restricted and users will see a "restricted access” message from the **Community** page.

To manage access to Community resources in the file browser:

1. From the file browser, click **Admin** in the left sidebar.
2. Select the **Settings** tab.
3. Scroll to **Resources** and select **Community Resources**.
4. In the **Customize Community visibility** modal, adjust the available options:
   - **File Browser**: Show or hide the Community tab in the [file browser](https://help.figma.com/hc/en-us/articles/14381406380183-Guide-to-the-file-browser).
   - **Product Template Pickers (Figma Buzz only)**: Show or hide Community resources (the “From Figma” tab) in [Figma Buzz template pickers](https://help.figma.com/hc/en-us/articles/31271689852823-Find-and-use-Figma-Buzz-templates).
5. Click **Save** to apply your changes.

### What happens when visibility is turned off?

- Users won’t see Community resources in the file browser or Community templates in Figma Buzz, depending on which settings you turn off.
- Users can still access the Community directly at [figma.com/community](https://www.figma.com/community).
