## 내용

This video is a recorded webinar produced outside of the structured lessons in this collection. It also provides a high level overview of the MCP server and how to use it. This video was produced when the Figma desktop MCP server was called the “Dev Mode MCP server”.

## About this video

In this replay of our August 19, 2025 webinar, Duncan Farquharson (Manager, Advocacy, Figma) and Oscar Watson-Smith (Enterprise Solutions Consultant, Figma) will walk you through:

- What an MCP server is
- Why you should build an MCP server for Dev Mode
- Design informed code generation
- Tips and tricks
- Dev Mode MCP server walk through
- Q&A
