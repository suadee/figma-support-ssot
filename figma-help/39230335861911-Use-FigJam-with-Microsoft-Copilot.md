## 내용

Before you start

Who can use this feature

Available on all Figma plans

You can connect Figma to Microsoft Copilot to generate structured FigJam diagrams directly from Copilot chats in Microsoft 365 apps.

When connected, Copilot can:

- Generate FigJam diagrams from prompts
- Turn documents into visual workflows
- Create structured diagrams like flowcharts, Gantt charts, sequence diagrams, and state diagrams

This integration currently supports FigJam only. You can’t use Copilot to create files for Figma Design, Figma Slides, or other Figma products.

## Connect Figma to Microsoft Copilot

To generate FigJam diagrams from Copilot, you’ll first need to connect your Figma account.

To connect Figma to Copilot:

1. [Open Microsoft Copilot](https://m365.cloud.microsoft/chat/).
2. Select **All agents** from the left sidebar.
3. Search for Figma.
4. Select the Figma app and click to **Add**.
5. Follow the authentication prompts to sign in and grant access.

Once connected, Copilot can generate FigJam files directly in response to your prompts.

## Generate FigJam diagrams in Copilot

After connecting Figma, you can ask Copilot to create diagrams directly from your conversation.

To generate a diagram:

1. Open Copilot in your Microsoft 365 app.
2. Enter a prompt describing the diagram you want to create. We recommend specifically mentioning `@Figma` in the prompt for best results.
3. Copilot generates a FigJam file and provides a preview or link.

You can use prompts like:

- “Create a flow chart for a user signup and login flow. Include account verification and two-factor authentication.”
- “Create a Gantt chart for a 6-week project with design, development, and QA sprints.”
- “Analyze this document and generate a process diagram in FigJam.”

Copilot analyzes your prompt and any referenced documents and creates a structured FigJam diagram with shapes, connectors, and labels.

The generated file is designed to be a structured starting point. You can continue refining layout, content, and visuals directly in FigJam.

## Open and edit the file in FigJam

When Copilot generates a diagram, it creates a new FigJam file in your Figma account and provides a link.

To continue editing:

1. Select the provided **Edit in Figma** or click the provided link.
2. If you’re signed in, the file appears in your drafts.
3. If you’re not signed in, log in or create a Figma account to claim the file.

Files that haven’t been claimed are public and viewable by anyone with the link. Once you log in and add the file to your drafts or a team, it becomes private according to your sharing settings.

Once created, the file behaves like any other FigJam file. You can:

- Move and edit objects
- Add sticky notes, shapes, and connectors
- Invite collaborators
- Share with your team

**Note:** We recommend using FigJam on desktop (app or browser). Editing isn’t supported on mobile.
