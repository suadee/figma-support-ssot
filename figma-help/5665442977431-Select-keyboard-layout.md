## 내용

Who can use this feature

Supported on [any team or organization plan](https://help.figma.com/hc/en-us/articles/360040328273).

Available in Figma Design and FigJam files.

Figma’s default keyboard shortcuts are based on the layout of a US QWERTY keyboard. Some of the keys in these shortcuts aren’t available in other languages or layouts.

You can update your keyboard layout in your preferences to access shortcuts mapped to that layout. Figma supports the following keyboard layouts:

- Chinese (Pinyin)
- Danish
- Finnish
- French AZERTY
- German QWERTZ
- Italian
- Japanese (Kana)
- Korean
- Norwegian
- Portugese
- Spanish
- Spanish (Latin America)
- Swedish
- U.K. (Mac / PC)
- U.S. QWERTY / Generic

If you don’t see your keyboard as an option, you can [submit a request using this form.](https://docs.google.com/forms/d/e/1FAIpQLSd-Pb0Ib3jNa0wn7cV3NVjZcTKOfQoYRuv5TVja103ym-ShHQ/viewform)

## Select keyboard layout

You can change your keyboard layout from any open file. Your preference will apply to your entire account. You can also use this setting to revert to Figma’s default keyboard layout.

**Note**: When possible, Figma will notify you of a mismatch between your system keyboard and your keyboard layout in Figma. If you do make a change to your system keyboard, make sure to also update your keyboard layout in Figma.

1. Click  in the bottom-right corner of your screen and select **Keyboard shortcuts**.
2. Select the **Layout** tab.
3. Use the dropdown to update your layout. Preview keyboard layouts by hovering over each option before selecting it.   
   - Select **Generic** to revert to the default keyboard layout and use Figma’s default mappings.
4. Use the tabs at the top of the panel to view the shortcuts for your layout. [Use shortcuts and quick actions →](https://help.figma.com/hc/en-us/articles/360040328653)

![Keyboard shortcuts item in help meny with panel expanded](https://help.figma.com/hc/article_attachments/31937331453463)

**Tip:** Access the keyboard shortcuts panel from the main menu. Open the  menu and go to **Preferences > Keyboard layout**.
