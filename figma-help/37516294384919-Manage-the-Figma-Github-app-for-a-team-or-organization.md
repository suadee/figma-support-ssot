## 내용

Who can use this feature

Available on [all plans](https://help.figma.com/hc/en-us/articles/360040328273)

Admins only

To support some features, Figma uses the Figma GitHub app. GitHub apps are integrations and tools provided in the GitHub Marketplace. The Figma GitHub app enables:

- Figma Make's push to GitHub feature. To let you back up your code, track changes, and continue working in your preferred development tools, you can [push your Make projects directly to GitHub](https://help.figma.com/hc/en-us/articles/35463818346647).
- Code Connect UI. [Code Connect UI](https://help.figma.com/hc/en-us/articles/23920389749655-Code-Connect#h_01K5CK9YP3CC33W1J1FGDWAC7W) lets you map design components in your Figma libraries to the corresponding code components in your GitHub account or organization repositories.

## Disconnect the Figma GitHub app

As a team or organization admin, you can disconnect the Figma GitHub app, which removes the ability to push from Figma Make to GitHub and connect Code Connect UI to your code components. If desired, you can connect the Figma GitHub app again later.

Starter planProfessional, Organization, or Enterprise plan

1. From the
   [file browser](https://help.figma.com/hc/en-us/articles/14381406380183),
   click
    **All projects**.
2. Click the
   next to the team name, and then select
   **View settings**.
3. Under **Figma GitHub app**, click **Manage**.
4. Click **Remove from Figma**.

1. Open your team or organization in Figma.
2. Select **Admin** from the sidebar.
3. Select the **Settings** tab.
4. In the **Resources** section, click
   **Figma GitHub app**.
5. Click **Remove from Figma**.

## Revoke authorization for the Figma GitHub app

After disconnecting the Figma GitHub app, you may want to fully revoke the authorization of the Figma GitHub app. When you revoke the authorization for the GitHub app, it means if you later want to reconnect the app, you'll also need to reauthorize it.

To revoke authorization for the Figma GitHub app:

1. Go to <https://github.com/settings/apps/authorizations>.
2. Click **Revoke** next to the Figma entry.
