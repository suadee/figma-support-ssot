## 내용

Productboard is a product management platform that helps organizations get the right products to market, faster. Product teams can use Productboard to understand what customers need, prioritize what to build next, and align stakeholders around the roadmap.

- **Insights**: Capture and collect all customer feedback and product ideas in a centralized repository
- **Prioritization**: Prioritize product initiatives based on customer data and standardized processes
- **Roadmap**: Align the company around a real-time, interactive roadmap
- **Portal**: Validate ideas, share roadmap plans, and announce launches through a public-facing portal

You can embed live Figma or FigJam files and Figma prototypes in the Productboard Editor UI.

You will need to [create a Productboard account](https://www.productboard.com/pricing/) to use this integration. This may require a separate paid subscription with Productboard.

## Embed a file or prototype in the Productboard Editor

You can embed multiple files or prototypes in a Productboard Editor field and expand them to view or interact in fullscreen.

You can embed content in any Editor UI found in the Insights, Feature, or Roadmap board. You will need [permission to edit content](https://support.productboard.com/hc/en-us/articles/1500012789762-How-to-manage-permissions-in-Productboard) in those boards.

Review l[ink sharing permissions](https://help.figma.com/hc/en-us/articles/360040531773-Share-Files-with-anyone-using-Link-Sharing#Update_Link_Sharing) for any files or prototypes you want to embed.

**Note:** If you set file access to **Only invited people**, you’ll need to log in to Figma before you can view the embed in Productboard.

1. Open the Figma file or prototype you want to embed.
2. Select the **Share** button in the toolbar.
3. Update [link sharing permissions](https://help.figma.com/hc/en-us/articles/360040531773-Share-Files-with-anyone-using-Link-Sharing#Update_Link_Sharing) to determine who can view and interact with the embed in Productboard. Click **Copy link.**
4. In Productboard, [open the Editor field](https://support.productboard.com/hc/en-us/articles/13223901654547-Embed-content-in-the-Productboard-Editor-) you would like to add an embed to.
5. Paste the URL in the Editor field. You can now [view the file or prototype](https://help.figma.com/hc/en-us/articles/360051741274) in the Productboard Editor.![Productboard interface showing a Figma embed in the design spec panel under Integrations for product ideation.](https://help.figma.com/hc/article_attachments/12806001441431)

**Tip:** To view or interact with the embed in full screen, select the expand icon in the upper right corner.

### Embed a specific frame

1. From the file or prototype, select the frame you'd like to embed.
2. Select the **Share** button in the toolbar.
3. Check and update l[ink sharing permissions](https://help.figma.com/hc/en-us/articles/360040531773-Share-Files-with-anyone-using-Link-Sharing#Update_Link_Sharing) as needed.
4. Check **Link to current selection** then click **Copy link**.
5. In Productboard, open the Editor field you would like to add an embed to.
6. Paste the URL in the Editor field.

You can now [view the file or prototype](https://help.figma.com/hc/en-us/articles/360051741274) in the Productboard Editor.

## Remove an embed from the Productboard Editor

To remove a Figma or FigJam embed from the Productboard Editor:

1. Select a text line below the embed.
2. Press the `Delete` or `Backspace` key to remove the embed.
