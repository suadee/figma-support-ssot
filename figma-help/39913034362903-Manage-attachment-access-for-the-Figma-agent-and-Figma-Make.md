## 내용

The Figma agent is currently available to try for free in open beta. [Learn more.](https://help.figma.com/hc/en-us/articles/34932042346775)

Who can use this feature

Available on [Organization and Enterprise plans](https://help.figma.com/hc/en-us/articles/360040328273)

[Organization admins](https://help.figma.com/hc/en-us/articles/4420557724439) only

Organization admins can control whether members can upload file attachments in the Figma agent and Figma Make. When attachments are enabled, members can add designs, images, video, audio, text files, and PDFs to their prompts. [Learn more about attaching files to a prompt.](https://help.figma.com/hc/en-us/articles/31304529835671)

### How it works

- If [AI features are disabled](https://help.figma.com/hc/en-us/articles/17725942479127#h_01J0SNQBZ9JXM4T45V5QWZ2QFZ) in your organization, file attachments aren't available in the agent chat.
- The file attachments setting applies to your entire organization. Workspace-level controls aren't available.
- When file attachments are disabled, members can't upload new files to their prompts and the option to add file attachments is removed from the agent chat. Any data from attachments already incorporated into a project will remain.
- File attachments are enabled for organizations by default.

## Enable or disable file attachments

1. From the [file browser](https://help.figma.com/hc/en-us/articles/14381406380183), click  **Admin**, then select  **Settings**.
2. In the **AI** section, toggle **Make file attachments** on or off.
