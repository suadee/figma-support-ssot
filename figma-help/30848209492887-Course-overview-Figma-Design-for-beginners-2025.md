## Course overview

This course was published in May 2025. Figma Design is always improving and some visual details shown in our videos may differ from the current version. Despite visual differences, the core functionality remains unchanged, so you can still follow along to achieve the same results.

Figma Design is used to create digital experiences of all kinds, including mobile apps, websites, illustrations, and more.

In Figma Design for beginners, you’ll discover how powerful designing in Figma can be. No matter your role or level of experience, if you’re looking for ways to turn your ideas into reality, this course is for you.

Throughout this hands-on course, we’ll design a portfolio website from scratch. Together, we’ll go step-by-step, from adding your first layers to the canvas, to building a fully responsive and interactive layout. We’ll cover the basics like shapes, text, and frames, and tap into more advanced features like auto layout, components, and prototyping. By the end, you’ll not only have a customizable portfolio website, you’ll also have the skills needed to start bringing your own ideas to life.

**Looking to learn in a different language?** This course uses YouTube's automatic dubbing feature to make the content available in different languages. You can change the language for a video by clicking ⚙️ **Settings** in the YouTube video player and selecting a language from the list. Visit YouTube's Help Center to learn more about [automatic dubbing with AI](https://support.google.com/youtube/answer/15569972?hl=en).

## Course requirements

To complete this course, you will need a computer, an internet connection, and a Figma account. The features we’ll be using are available on the free [Starter plan](https://help.figma.com/hc/en-us/articles/360040328273-Figma-plans-and-features), so you don’t need a paid subscription to get started. If you don’t have a Figma account, we’ll walk through how to create one in the next video.

You can use Figma in the browser or download the [Figma desktop app](https://www.figma.com/downloads/). Learn more about Figma's [system requirements](https://help.figma.com/hc/en-us/articles/360039827194-What-are-the-system-requirements-for-Figma).
