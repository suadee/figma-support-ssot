Who can use this feature

Available on [any plan](https://figma.zendesk.com/hc/en-us/articles/360040328273-Choose-a-Figma-Plan)

Anyone with `can edit` access to a file can edit an object’s motion path

When you apply position keyframes to an object, the movement is displayed as a motion path on the canvas. A motion path appears as a sequence of dots indicating an animation's [easing](https://help.figma.com/hc/en-us/articles/41414048690839-Adjust-an-animation-s-easing) and boxes marking the position of each [keyframe](https://help.figma.com/hc/en-us/articles/41307938657559-Add-select-and-delete-keyframes).

## Edit a motion path

You can edit the motion path to change how an object moves. An object’s motion path can only be edited in  **Figma Motion**.

1. Select an object with position keyframes applied to reveal its motion path.
2. Do one of the following:
   - **To reposition a keyframe**: Click and drag it to a new position on the canvas
   - **To curve a motion path:** Hold `Command` (Mac) or `Control` (Windows), then select a keyframe and use the handle to change the path’s curve

![](https://help.figma.com/hc/article_attachments/41780289462423)

**Note:** An animation’s easing also influences how an object moves. To adjust an animation’s easing, select the animation on the timeline or click the line between two keyframes, then choose an option from the Easing menu. Learn more about [easing](https://help.figma.com/hc/en-us/articles/41414048690839-Adjust-an-animation-s-easing#h_01KVRPDGBXKK58JKZGB3TJ4RN2).
