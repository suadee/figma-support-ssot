## 내용

Who can use this feature

Available on all plans

Your team or organization’s **AI toggle** must be enabled to access Figma Make. [Learn more.](https://help.figma.com/hc/en-us/articles/17725942479127-Manage-AI-settings-and-content-training-for-your-team-or-organization)

Figma Make gives you access to the latest AI models through the model selector, so you can see how different models handle prompts, generate creative interactions, and interpret design context. Figma Make has a default model and a number of other models. Exploring different models allows you to compare results and discover new capabilities of AI.

The following models are currently available for use in Figma Make, in addition to the default model:

|  |  |
| --- | --- |
| **AI model** | **Try this model for** |
| Gemini 3 Flash | Faster and more iterative process |
| Gemini 3.1 Pro | More complex and creative work |
| Claude Sonnet 4.6 | A balance of versatility and efficiency |
| Claude Opus | Challenging projects and deep research |
| GPT-5.6 | Capable and quick iterations |

Using different models doesn’t change how you work in Figma Make—the menus, chat panel, and code view all remain the same.

**Note**: Non-default models may sometimes exhibit different behaviors than the default model, including unexpected errors, additional self-healing, or longer reasoning and generation.

Credit consumption for non-default models is still in development and may vary significantly. Some models may use substantially more credits per task than other available options.

## **Change which AI model you're using in Figma Make**

You can choose which model to use when chatting in Figma Make:

1. Open a new or existing Figma Make file
2. In the chat box, you’ll see which AI model you’re currently using
3. Click on the name of the model to open a dropdown of model options
4. Select your preferred model to use for this prompt

## Frequently asked questions

**Can I switch models during a conversation with Figma AI?**

Yes, you can switch models at any time while you’re chatting in Figma Make.

When you create a new Figma Make file, it starts with the default model selected. If you return to an existing file, Figma Make uses the model you last selected for that file.

**What is the default model for Figma Make?**

The default model may change over time. We regularly update the model to ensure Figma Make users get the best performance.

**Will Figma add more models in the future?**

We continually test the latest models and plan to introduce more options in the future. Stay tuned!

**How many AI credits do different models use?**

Agentic AI features run on underlying language models. Figma sets pricing for both the model and the feature it’s used in based on factors like the model’s cost to Figma, the value it provides to customers, and the consumption patterns of that model. Credit consumption can also vary based on the complexity of the task and the volume of context that the model processes to complete the task successfully.

Your own credit history is the best way to understand how different models may consume credits.

Learn more about [how AI credits work](https://help.figma.com/hc/en-us/articles/33459875669015).
