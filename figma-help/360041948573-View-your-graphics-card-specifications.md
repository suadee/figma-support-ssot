## 내용

**What:** View and export information about your Graphics Processing Unit (GPU) using Chrome's built-in tools.

**Why:** This report lets us know important information about your GPU. This includes which version you're running, color space, status and what specific features are currently enabled.

# View your graphics card specifications

1. Open the Chrome browser.
2. Enter `chrome://gpu/` in the address bar:

   ![Chrome browser with "chrome://gpu/" URL entered in the address bar for accessing GPU specifications.](https://s3.amazonaws.com/helpscout.net/docs/assets/5aa962fe2c7d3a2c4983093d/images/5d7732152c7d3a7e9ae0f201/file-tStabO8Ml1.png)
3. This will give you a detailed report of your GPU specifications:

   ![Chrome browser displaying GPU specifications, showing hardware acceleration status and driver bug workarounds.](https://s3.amazonaws.com/helpscout.net/docs/assets/5aa962fe2c7d3a2c4983093d/images/5d77321b04286364bc8ee9bd/file-vRGXn4yF8d.png)
4. Right-click on the screen and choose **Save as**:

   ![Chrome's GPU report with right-click menu open, highlighting "Save As" option for exporting detailed GPU specifications.](https://s3.amazonaws.com/helpscout.net/docs/assets/5aa962fe2c7d3a2c4983093d/images/5d7732252c7d3a7e9ae0f202/file-WKsu7nByMG.png)
5. Ensure the **Format** is set to "Webpage Complete" before you click **Save**:

   ![Chrome GPU specifications page with a Save window open.](https://s3.amazonaws.com/helpscout.net/docs/assets/5aa962fe2c7d3a2c4983093d/images/5d7732812c7d3a7e9ae0f208/file-wx4731cqj8.png)
6. Attach the saved file to your Figma Support Conversation.
