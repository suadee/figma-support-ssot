## 내용

The Figma integration with Zapier connects Figma to thousands of other tools without writing code. The integration helps teams stay focused on building great products by automating routine tasks, like sending notifications when files are updated, logging design feedback, and creating tickets for follow up tasks.

Example use cases:

- Keep teams updated: Post a Slack message when a new file version is published in Figma.
- Track design progress: Log file updates in Airtable or Google Sheets for project tracking.
- Close the feedback loop: Create Jira or Asana tasks when new comments are added to a file.
- Manage developer handoff: Add dev resources to your workflow tool when they’re created in Figma.
- Streamline QA: React to comments automatically or move them into Trello when action is required.

**Note:** Need help with Zapier? [Check out their help center.](https://help.zapier.com/hc/en-us)

## Set up the Figma and Zapier integration

To create an app connection to Figma on Zapier:

1. Go to the Apps page.
2. Click **+ Add** connection.
3. A new dialog box will appear. Search for and select Figma.
4. Click **Add connection**.
5. A new browser tab or window will open.
6. Log into Figma to authenticate.
7. Grant Zapier permission to access your account if prompted to.

[Learn more about setting up this integration.](https://help.zapier.com/hc/en-us/articles/39640412485773-How-to-get-started-with-Figma-on-Zapier)

## Setting up triggers or actions from Figma within a Zap

Zapier helps you build automatic workflows by connecting triggers and actions. You can set an action from Figma to be used as a trigger or an action.

Learn how to set up a Zapier [trigger](https://help.zapier.com/hc/en-us/articles/8496288188429-Set-up-your-Zap-trigger) or [action](https://help.zapier.com/hc/en-us/articles/8496257774221-Set-up-your-Zap-action).

### Available Figma triggers

- New Comment – triggers when a new comment is added
- New File Version – triggers when a new version of a file is created
- New Dev Resource – triggers when a new developer resource is added
- File Updated (instant) – triggers instantly when a file is updated
- New Comment (instant) – triggers instantly when a new comment is added

### Available Figma actions

- Create Image – create an image from a file
- Create Comment – add a new comment to a file
- Delete Comment – remove an existing comment
- React to Comment – add a reaction to a comment
- Delete Comment Reaction – remove a reaction from a comment
- Create Dev Resource – add a new developer resource
