## 내용

Who can use this feature

Supported on [any team or plan](https://help.figma.com/hc/en-us/articles/360040328273)

You can use Figma on Chrome, Safari, Microsoft Edge and Firefox

You need WebGL installed and enabled to use Figma. This website will let you know if this is enabled: <http://webglreport.com/>

There are a couple of things you'll want to set up on your browser, before getting started with Figma.

- Confirm you have WebGL installed and enabled
- Set your browser zoom to 100%
- Make sure you have the latest browser version
- [Install the Figma Font Helper to use fonts installed on your computer](https://help.figma.com/hc/en-us/articles/360039956894)
- [Disable the left/right swipe gesture](#mac) (Mac only)

**Using the Figma desktop app?** You don't have to update these settings to use the desktop app. You can still update these settings if you plan on using Figma in the browser in the future.

## Update settings

Select your browser below for more detailed instructions.

Google Chrome Mozilla Firefox Safari Microsoft Edge

1. Open Chrome and go to the **Settings** page `chrome://settings/`.
2. In the **Appearance** section, ensure your **Page zoom** is set to 100%. This will ensure your files are displayed accurately at higher resolutions.![Google Chrome settings showing "Page zoom" set to 100% for Figma display.](https://d33v4339jhl8k0.cloudfront.net/docs/assets/5aa962fe2c7d3a2c4983093d/images/5bae5c3b2c7d3a04dd5b187a/file-L2DVRAzUF7.png)
3. In the **System** section, enable **Use graphics acceleration when available**
4. If WebGL isn't enabled, open Chrome's **Flags** settings `chrome://flags/`.
5. Update the **Override software rendering list** to enabled. This should be near the top of the list: ![Chrome settings panel showing the "Override software rendering list" option with "Disabled" selected.](https://d33v4339jhl8k0.cloudfront.net/docs/assets/5aa962fe2c7d3a2c4983093d/images/5bae7141042863158cc6e0ec/file-9dgTUrDPGh.png)
6. Alternatively, you can search for WebGL related flags and update this to **Enabled**.

1. Open Firefox and open the menu in the top-right corner.
2. This will allow you to see the current **Zoom** level. You can then use the `-` and `+` icons to decrease/increase the zoom level:
3. From that same menu, select **Preferences** from the options:  
   ![Firefox menu highlighting the Preferences option for adjusting settings relevant to using Figma.](https://d33v4339jhl8k0.cloudfront.net/docs/assets/5aa962fe2c7d3a2c4983093d/images/5bae7007042863158cc6e0db/file-MTfKkeVty2.png)
4. Scroll down to the **Firefox Updates** section. You'll be able to see the current version and whether or not it is up to date:

   ![](https://d33v4339jhl8k0.cloudfront.net/docs/assets/5aa962fe2c7d3a2c4983093d/images/5bb20f47042863158cc6ec28/file-91eVqDcdPE.png)
5. You can click the **Check for updates** button to check for any updates. If you're not on the latest version this will prompt an install of the latest version. Firefox will automatically restart after installing.
6. We recommend updating the Allow Firefox to setting to **Automatically install updates**.

Safari is automatically updated by macOS whenever there is a new version. If there is a new version of macOS available, you'll be able to update this on your computer directly, which will also update Safari with any required updates.

1. Open Figma in Safari.
2. Head to **Safari** > **Preferences** in the main menu.
3. Go to the **Websites** tab and click **Page Zoom**.
4. In the Currently open website section, ensure Figma is set to 100%: ![Safari settings showing Figma.com page zoom set to 100%.](https://d33v4339jhl8k0.cloudfront.net/docs/assets/5aa962fe2c7d3a2c4983093d/images/5baec59a2c7d3a04dd5b1c9a/file-OuVw4lpURo.png)

**Note:** You'll need to be using at least V10.11.1 (El Capitan) if you want to use pinch-and-zoom functionality.

Figma no longer supports Internet Explorer: **[Explore supported browsers →](https://help.figma.com/hc/en-us/articles/360039827194)**

To update Microsoft Edge to the latest version:

1. From the **Start** menu select **Settings** > **U****pdates & Security**.
2. Select **Windows updates** at the top of the list and then **Check for updates**.
3. If there are any available, they will start downloading. Microsoft Edge will be updated and restarted automatically.

To set the default zoom:

1. Select **Menu** in the top-right corner of the window.
2. Select **Settings** from the options.
3. Select **Appearance** from the menu in the left column.
4. In the **Zoom** section, update the **Page zoom** setting to choose a new default zoom level.

### Enhanced Security

To use Figma on Microsoft Edge, you'll need to make sure the **Strict Mode** setting is disabled for `Figma.com`. Visit [Microsoft's help center](https://docs.microsoft.com/en-us/deployedge/microsoft-edge-security-browse-safer) for information.

## Update OS settings

If you're using the trackpad on macOS, then you may want to disable the left/right swipe gesture. This will prevent you from accidentally closing out of the Figma file, when you are scrolling left and right within the canvas.

1. Open **Systems preferences** and go to the **Trackpad** settings: ![macOS System Preferences window highlighting Trackpad settings for swipe gesture adjustments.](https://d33v4339jhl8k0.cloudfront.net/docs/assets/5aa962fe2c7d3a2c4983093d/images/5bb20fc2042863158cc6ec2e/file-p0hUhjXmrM.png)
2. Go to the **More Gestures** tab: ![Trackpad settings on macOS showing gesture options, highlighting the "Swipe between pages" feature in the "More Gestures" tab.](https://d33v4339jhl8k0.cloudfront.net/docs/assets/5aa962fe2c7d3a2c4983093d/images/5bb20fbf042863158cc6ec2d/file-xTTyZeOqeb.png)
3. Disable the **Swipe between pages** option: ![Trackpad settings on macOS showing the option to disable 'Swipe between pages' highlighted.](https://d33v4339jhl8k0.cloudfront.net/docs/assets/5aa962fe2c7d3a2c4983093d/images/5bb20fd32c7d3a04dd5b24d4/file-1NG4k1mLyb.png)
