## 내용

Before you Start

Who can use this feature

 

Available on [any plan](https://help.figma.com/hc/en-us/articles/360040328273-Choose-a-Figma-Plan)

Anyone with `can edit` access to a slide deck can add shapes with text

Use shapes with text to enhance your presentations by adding diagrams, flowcharts, or user journey flows to your slides.

## Add shapes with text

You can access the shapes with text tools from the toolbar while in slides mode. Unlike shapes added in design mode, shapes with text have a text layer built directly into the object.

1. Click  **Shapes** in the toolbar.
2. Select the shape you want to add.
3. Click on a slide to place the shape.

Once created, double-click inside the shape to start typing. You can use the settings in the right sidebar to change the shape and text properties.
