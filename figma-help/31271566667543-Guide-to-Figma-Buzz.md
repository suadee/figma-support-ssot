## 내용

Before you start

Who can use this feature

While in beta, Figma Buzz is available on all seats and plans.

Publishing templates is available on all paid plans. Professional teams can create five team templates.

![Cursor adjusts ad size from Pinterest to Instagram, changes background color, and adds arrow image to financial ad.](https://help.figma.com/hc/article_attachments/31913627802647)

## What is Figma Buzz

Buzz enables teams to create on-brand static assets together, with the power of Figma Design.

Start from a template or from scratch to make assets in Buzz, for use cases like:

- **Social media posts**: Help your brand stand out on social with templates for multiple platforms, sizes, and organic versus paid.
- **Digital ads**: Figma Buzz has pre-made templates and blank frames for ads across platforms. You can also create ads using your brand team’s branded templates.
- **Promotions**: Create digital fliers, email header images, and other promos.
- **Event materials**: Make all your event assets in Figma Buzz, from wayfinding to badges and swag.
- **Team communications**: From office signage to benefits infographics, creating internal comms assets is easier in Buzz.
- **Celebrations**: Say thanks with flair. Make birthday cards, wedding invites, thank you notes, and other personal event designs.

## Create a Buzz file

### File browser

1. Open the team or organization space you wish to create your asset.
2. Click **+ Create** then select **Buzz**.

You can also create a new file from a team folder.

### Link

Type [`figma.com/buzz/new`](https://figma.com/buzz/new) in the address bar.

Access Figma Buzz-specific file templates to get you started.

## Publish templates in Buzz

You can create templates in Figma Buzz using the built-in creation tools. If you're already working in Figma Design, you can use existing designs as a starting point — copy your frames or components, paste them into a Buzz asset, make any necessary edits, and publish the file as a template from the Share modal.

[Learn how to publish templates in Figma Buzz](https://help.figma.com/hc/articles/31271693614487).

## Edit assets

Figma Buzz makes it easy for you to create and share on-brand assets, so your teammates can grab what they need, when they need it.

### Editing a template with guidelines

When your team creates a template with guidelines, it sets boundaries for what can be edited—so you can make changes confidently without accidentally breaking the design. You can tell an asset has guidelines if it has a pink outline on the canvas and a lock icon next to the asset name.

When working with a template with guidelines, you can edit any unlocked fields from [icon] Table view. Tools like Text, Images, Shapes, and Inserts won't be available—only the fields your template author has approved for editing.

[Learn more about using templates](https://help.figma.com/hc/en-us/articles/31271689852823).

### Editing without guidelines

If an asset has a blue outline, it doesn't have guidelines — you have full access to all of Figma Buzz's creation tools, including text, images, shapes, and inserts. If you have a Full seat on a paid plan, you can also [use Figma Design tools in Figma Buzz](https://help.figma.com/hc/articles/31271851622807).

[Learn how to edit assets in Figma Buzz](https://help.figma.com/hc/articles/31271798141975).

## Bulk create assets

Use bulk create in Figma Buzz by uploading .csv and .xlsx spreadsheets. When you upload a spreadsheet, you can assign layers in the asset to columns of data in the spreadsheet. Once your layers and data are assigned and connected, Figma Buzz will create a new asset with each set of text or image values in your data.

[Learn how to bulk create assets in Figma Buzz](https://help.figma.com/hc/articles/31271824185623)
