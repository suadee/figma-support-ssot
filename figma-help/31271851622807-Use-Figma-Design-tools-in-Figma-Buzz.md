## 내용

Figma Buzz was released in [open beta](https://help.figma.com/hc/en-us/articles/4406787442711) at Config 2025. It is currently available on all plans and seat types. [Learn more about what’s included in the beta.](https://help.figma.com/hc/en-us/articles/4406787442711)

Figma Buzz lets you access Figma Design features such as auto layout, the layers panel, and advanced layer properties from the canvas.

## Enable design mode

If you have a **Full seat** and `can edit` access to the Figma Buzz file, you can access Figma Design tools using the  toggle in the toolbar or with the keyboard shortcut `Shift` `D`.

![Top: Basic toolbar with move and comment tools. Bottom: Expanded design mode with additional shape and text tools.](https://help.figma.com/hc/article_attachments/35629584995863)

## Layers panel

When using Figma Design tools in Figma Buzz, you’ll have access to the **Layers** panel in the lower right corner of the canvas. From there, you can [view and manage layers](https://help.figma.com/hc/en-us/articles/360039831974-View-layers-and-pages-in-the-left-sidebar#layers) for objects in an asset as you would in Figma Design.

## Right sidebar

The right sidebar gives you access to the **Design** tab. While in  **single asset view**, the right sidebar is always visible. If you’re using  **grid view**, the right sidebar only displays when you have an asset selected.

When you select an objects layer in Figma Buzz, you’ll have access to the advanced properties available for that layer type, including:

- [Alignment, rotation, and position](https://help.figma.com/hc/en-us/articles/360039956914-Adjust-alignment-rotation-and-position)
- [Frame size and orientation](https://help.figma.com/hc/en-us/articles/360041539473-Frames-in-Figma)
- [Layout grids](https://help.figma.com/hc/en-us/articles/360040450513-Create-layout-grids-with-grids-columns-and-rows)
- [Component properties](https://help.figma.com/hc/en-us/articles/5579474826519-Explore-component-properties)
- [Component and instance swap](https://help.figma.com/hc/en-us/articles/360039150413-Swap-components-and-instances)
- [Auto layout](https://help.figma.com/hc/en-us/articles/360040451373-Explore-auto-layout-properties)
- [Blend modes](https://help.figma.com/hc/en-us/articles/360040667874-Use-blend-modes-to-create-unique-effects)
- [Text properties](https://help.figma.com/hc/en-us/articles/360039956634-Explore-text-properties) (including [OpenType features](https://help.figma.com/hc/en-us/articles/4913951097367))
- [Fill](https://help.figma.com/hc/en-us/articles/360040623954-Add-fills-to-text-and-shape-layers)
- [Stroke](https://help.figma.com/hc/en-us/articles/360049283914-Apply-and-adjust-stroke-properties)
- [Layer effects](https://help.figma.com/hc/en-us/articles/360041488473-Apply-shadow-or-blur-effects)
- [Export settings](https://help.figma.com/hc/en-us/articles/24848334599447-Export-from-Figma-Slides)
