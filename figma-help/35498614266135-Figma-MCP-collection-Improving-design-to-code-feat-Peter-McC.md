## 내용

This video is a pre-recorded webinar produced outside of the structured lessons in this collection. If you’re looking for a glimpse into the team behind the MCP server, this will be a fun watch for you! This video was produced when the Figma desktop MCP server was called the “Dev Mode MCP server”.

## About this video

The Figma MCP server enables users to extract critical context information from Figma that helps inform better AI code generation.

In this webinar, Yarden Katz (Product Manager for MCP) and Peter McCarron (PMM for Dev Tools) walk through some best practices for how to structure your design files to ensure better design to code translation, use annotations and rules to align outputs with your codebase, and provide other tips & tricks for optimizing your MCP outputs.
