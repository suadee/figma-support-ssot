## 내용

Before you start

Who can use this feature

Supported on [Professional, Organization, and Enterprise plans](https://help.figma.com/hc/en-us/articles/360040328273)

Only the team owner or a team admin can claim ownership of a project.

If there is no longer an owner of a project, the team owner or a team admin can join the project as the owner.

This applies to:

- Projects shared with the entire team without an owner
- Private projects without an owner
- View-only projects without an owner

## Claim ownership of a Project

1. Navigate to the project in the file browser.
2. Click the **Join as Owner** button.

Once you have claimed the project, you can:

- [Move the project between teams](https://help.figma.com/hc/en-us/articles/360038511573)
- [Transfer ownership of the project](https://help.figma.com/hc/en-us/articles/360038512093)
- [Delete the project](https://help.figma.com/hc/en-us/articles/360038511833)

**Note:**If the team also has no owner, an organization admin or workspace admin can [claim ownership of the team](https://help.figma.com/hc/en-us/articles/360039481134).
