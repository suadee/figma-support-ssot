## 내용

Before you start

Who can use this feature

Available on the [Starter](https://help.figma.com/hc/en-us/articles/360040328273) and [Professional](https://help.figma.com/hc/en-us/articles/360040328273) plans

Anyone who is a member of the team can star files and projects they have access to

Star your most important files and projects to quickly access them from the sidebar of the file browser. Adding or removing starred items in your sidebar won’t change their location in Figma and only you can see which items you’ve favorited.

## Star a file

Once you’ve starred a file, you can find it under the **Starred** section in the left sidebar of the [file browser](https://help.figma.com/hc/en-us/articles/14381406380183).

### Star a file from the file browser

- Hover over a file and click the  **star icon**.
- Right-click a file, then select **Add to your favorites**.
- Drag a file directly into the **Starred** section.

### Star a file from the canvas

With a file open, click the  **dropdown menu** next to the file name in the toolbar, then select **Add to sidebar**.

## **Star a project**

When you star a project, it will appear under its parent team in the left sidebar. There are a few ways to star a project:

### Star a project from the team page

1. From the [file browser](https://help.figma.com/hc/en-us/articles/14381406380183), click the team name in the left sidebar.
2. Find the project in your list of projects.
3. Hover over the project tile and click the  **star icon**.

### Star a project from the project page

1. Open the project in the file browser.
2. Click the  **star icon** in the top-right corner to favorite it.

## Organize or remove starred items

You can click and drag files you’ve starred to reorder them in the left sidebar.

To remove a starred item:

1. Right-click the file or project name from the left sidebar.
2. Select **Remove from sidebar**.
