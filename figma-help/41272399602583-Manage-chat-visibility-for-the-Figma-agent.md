## 내용

Starting June 23, 2026, new chats with the agent in Figma Design are visible by default to people with a Full seat and edit access to the file. Chats created before June 23, 2026 remain private unless you make them public.

Who can use this feature

Available on [Professional, Organization, and Enterprise plans](https://help.figma.com/hc/en-us/articles/360040328273)

Requires access to the [Figma agent](https://help.figma.com/hc/en-us/articles/37998629035799) in design files

People with a Full seat and edit access to the file can view public chats in that file

When you work with the [Figma agent in a design file](https://help.figma.com/hc/en-us/articles/37998629035799), your conversations can help collaborators understand how a result was created, what you asked the agent to do, and what context informed the output.

New chats are visible in the file by default, which means anyone in your team or organization with edit access to the file can see them.

You can make a chat private at any time if it includes sensitive information, unfinished exploration, or context you do not want others in the file to see.

## Change who can access a chat

![image (4).png](https://help.figma.com/hc/article_attachments/41439220164247)

You can change the visibility of chats you created using the below access settings. You cannot change the visibility of someone else's chat.

When you make a chat private, it is removed from other collaborators' chat lists.

| Access | Who can view | Who can continue the chat | Use this for |
| --- | --- | --- | --- |
| Shared chat | You, and people with a Full seat and edit access to the file | Only the person who created the chat; everyone else can view the chat, but can’t continue it | Workflows, prompts, and decisions that are useful for collaborators to understand |
| Private chat | Only you | Only you | Sensitive work, early exploration, personal notes, or chats that should not be visible to collaborators |

To change access to a chat:

1. Click  **Agents** in the left navigation bar.
2. Open the chat you want to update.
3. Click  **More options** in the chat’s header.
4. Set the chat access to **Shared chat** or **Private chat**.

**Tip:** You can also change the visibility of all your existing chats in the file in one go.

1. Click  **Agents** in the left navigation bar.
2. From the **Chats** view, click  **More options** and then select **Make your existing chats private**.

**Note:** Changing access to a chat applies to the whole conversation. You cannot make individual prompts or messages private within a chat.

## Share a link to a chat

You can copy a link to a chat when you want to point a collaborator to the exact conversation behind a result or decision.

When someone opens the link, Figma checks whether they have access to the file and permission to view the chat. A link to a public chat can be opened by people with a Full seat and edit access to the file.

To share a chat:

1. Open  **Agents** in the left navigation bar.
2. Open the chat you want to share.
3. Click  **More options** in the chat’s header, then select **Chat access** > **Copy link to chat**.
4. Share the link with a collaborator.

**Note**: You can't share a link to a private chat.

## Frequently asked questions

### Can I make new chats private by default?

New chats with the agent in Figma Design are public by default starting June 23, 2026. You can make individual chats private after you create them.

### Can people with view access see the file's chats?

No. Chats are only visible to people with a Full seat and edit access to the file.
